⚠️ PROBLÈME DÉTECTÉ
--------------------
Un fichier de verrouillage Git (.git/index.lock) bloque les opérations.
Cela arrive quand un processus Git précédent n'a pas terminé correctement.

✅ SOLUTION MANUELLE (RECOMMANDÉE)
----------------------------------

ÉTAPE 1: Supprimer le fichier de verrouillage
----------------------------------------------
Ouvrez un nouveau terminal PowerShell et exécutez:

Remove-Item -Path "H:/ClaraVerse/.git/index.lock" -Force

ÉTAPE 2: Vérifier qu'aucun processus Git ne tourne
---------------------------------------------------
Get-Process | Where-Object {$_.ProcessName -like "*git*"}

Si des processus Git apparaissent, fermez-les:
Stop-Process -Name "git" -Force

ÉTAPE 3: Configurer Git
------------------------
git config core.compression 0
git config http.postBuffer 1048576000
git config http.lowSpeedTime 999999
git config http.lowSpeedLimit 0

ÉTAPE 4: Changer le repository distant
---------------------------------------
git remote set-url origin https://github.com/sekadalle2024/Claverse_windows__v8_17-04-2026_V5-Export_CAC-V0-Public.git

ÉTAPE 5: Vérifier la branche
-----------------------------
git branch

ÉTAPE 6: Ajouter tous les fichiers
-----------------------------------
git add .

ÉTAPE 7: Créer le commit
-------------------------
git commit -m "Sauvegarde ClaraVerse - Version 8 - 17 Avril 2026"

ÉTAPE 8: Push vers GitHub
--------------------------
git push -u origin master

(ou "main" si votre branche s'appelle "main")
