📋 ÉTAPES RAPIDES:

# 1. Générer clé SSH (si nécessaire)
ssh-keygen -t ed25519 -C "votre-email@example.com"

# 2. Copier la clé
Get-Content ~/.ssh/id_ed25519.pub | Set-Clipboard

# 3. Ajouter à GitHub
https://github.com/settings/keys → New SSH key → Coller

# 4. Tester
ssh -T git@github.com

# 5. Changer l'URL
git remote set-url origin git@github.com:sekadalle2024/Claraverse_windows_s_11_v0_28-03-2026_V5-Export_CAC-V0-Public.git

# 6. Configuration optimale
git config core.compression 0
git config pack.windowMemory "100m"
git config pack.packSizeLimit "100m"
git config pack.threads "1"

# 7. Push
git push -u origin master --verbose

⏱️ Temps total: 15-20 minutes
