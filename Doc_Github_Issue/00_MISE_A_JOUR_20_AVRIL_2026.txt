═══════════════════════════════════════════════════════════════════════════════
  CONFIGURATION GIT APPLIQUÉE
═══════════════════════════════════════════════════════════════════════════════

git config core.compression 0
git config http.postBuffer 2147483647
git config http.lowSpeedTime 999999
git config http.lowSpeedLimit 0
git config http.maxRequestBuffer 536870912

Raison:
  • Désactiver la compression → Évite les timeouts
  • Augmenter buffer HTTP à 2 GB → Gère les gros fichiers
  • Désactiver limites de vitesse → Pas d'interruption
  • Augmenter taille max requêtes → Requêtes plus grandes
