# 📖 Contexte Détaillé - Sauvegarde ClaraVerse V5

## 🎯 Vue d'Ensemble

### Projet ClaraVerse
ClaraVerse est une application d'audit et de révision comptable basée sur React/TypeScript avec un backend Python utilisant LangGraph.

### Historique du Projet
1. **Origine**: Projet open source cloné depuis GitHub
2. **Modifications**: Améliorations et personnalisations importantes
3. **Besoin**: Sauvegarder la version modifiée dans un nouveau repository

---

## 🏗️ Architecture du Projet

### Frontend
- **Framework**: React avec TypeScript
- **Composants principaux**:
  - DemarrerMenu.tsx
  - CyclesComptablesSidebar.tsx
  - Clara_Components/
- **Services**: claraApiService.ts
- **Intégrations**: N8N, Flowise

### Backend
- **Langage**: Python
- **Framework**: LangGraph (microservices)
- **Containerisation**: Docker
- **Fichiers clés**:
  - py_backend/Dockerfile
  - py_backend/README.md

### Documentation
- **Manuel E-audit**: Documentation complète pour auditeurs
- **Guides de déploiement**: Netlify, Koyeb
- **Documentation N8N**: Intégration workflows
- **Doc_AIONUI**: Composants UI personnalisés
