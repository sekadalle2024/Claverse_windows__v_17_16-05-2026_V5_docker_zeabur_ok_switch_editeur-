🎓 COMPRENDRE LE PROBLEME

Vous avez essayé:
curl -s "https://api.github.com/repos/..." | ConvertFrom-Json

PowerShell a demandé: "Uri:"

POURQUOI?
- `curl` = alias de `Invoke-WebRequest`
- Nécessite paramètre `-Uri` explicite
- La commande manuelle ne fonctionne pas

SOLUTION:
Utiliser le script automatique qui gère tout:
& ".\Doc render deploy\verifier-fichiers-github.ps1"
