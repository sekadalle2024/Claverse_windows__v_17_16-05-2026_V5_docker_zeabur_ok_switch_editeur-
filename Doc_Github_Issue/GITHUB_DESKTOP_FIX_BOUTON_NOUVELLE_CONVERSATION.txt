📝 DESCRIPTION DU COMMIT
-----------------------------------------------------------------
🐛 Problème résolu:
Le bouton "Nouvelle conversation" (New Chat) dans la barre latérale 
gauche n'apparaissait pas ou était masqué par les scripts de l'interface.

✅ Solution appliquée:

1. Modification de design_chat.js
   - Exception ajoutée pour préserver le bouton "New Chat"

   - Force l'affichage du bouton
   - Injecte des styles CSS
   - Observe le DOM pour réappliquer automatiquement

3. Intégration dans index.html
   - Script chargé après design_chat.js
