🔍 SYMPTÔME
─────────────────────────────────────────────────────────────────────────────
Page blanche après redémarrage de l'application

🎯 CAUSE PROBABLE
─────────────────────────────────────────────────────────────────────────────
Erreur de compilation React ou erreur dans le composant NotificationContainer

⚡ DIAGNOSTIC IMMÉDIAT
─────────────────────────────────────────────────────────────────────────────

ÉTAPE 1 : OUVRIR LA CONSOLE
────────────────────────────
F12 → Onglet Console

ÉTAPE 2 : CHERCHER L'ERREUR
────────────────────────────
Chercher des messages en rouge commençant par :
• "Error:"
• "Uncaught"
• "Failed to compile"
• "SyntaxError"

ÉTAPE 3 : VÉRIFIER LE TERMINAL
───────────────────────────────
Dans le terminal où tourne npm run dev, chercher :
• "ERROR"
• "Failed to compile"
• Ligne rouge

📋 ERREURS POSSIBLES
─────────────────────────────────────────────────────────────────────────────

ERREUR 1 : Import CSS manquant
───────────────────────────────
Message : "Cannot find module './NotificationContainer.css'"
Solution : Le fichier CSS existe, vérifier le chemin

ERREUR 2 : Syntaxe JSX
──────────────────────
Message : "Unexpected token" ou "JSX element"
Solution : Vérifier la syntaxe dans NotificationContainer.tsx

ERREUR 3 : Hook React
─────────────────────
Message : "Invalid hook call"
Solution : Vérifier que les hooks sont bien utilisés
