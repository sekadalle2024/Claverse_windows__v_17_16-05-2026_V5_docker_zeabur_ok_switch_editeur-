📋 CONTEXTE
-----------
Suite aux dommages système et restauration depuis Github, la CLI Koyeb
n'est plus installée. Les paramètres de configuration existent toujours.
Nous devons réinstaller la CLI et mettre à jour le déploiement backend.

🎯 OBJECTIFS
------------
1. Réinstaller la CLI Koyeb sur Windows
2. Vérifier la configuration existante
3. Mettre à jour le déploiement du backend Python (py_backend)

⚡ DÉMARRAGE ULTRA-RAPIDE
--------------------------
Exécuter dans l'ordre :

1. .\install-koyeb-cli.ps1
2. koyeb login
3. cd py_backend
4. koyeb deploy . clara-backend/clara-service --archive-builder docker
