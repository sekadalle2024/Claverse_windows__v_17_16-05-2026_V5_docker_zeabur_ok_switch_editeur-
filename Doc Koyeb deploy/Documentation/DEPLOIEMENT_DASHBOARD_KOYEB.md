# 🌐 Déploiement via Dashboard Koyeb (Sans CLI)

## ✨ Méthode la Plus Simple pour Windows

Cette méthode ne nécessite **aucune installation de CLI**. Tout se fait via l'interface web.

## 🚀 Étapes de Déploiement

### Étape 1 : Créer un Compte Koyeb

1. Aller sur : **https://app.koyeb.com/auth/signup**
2. S'inscrire avec :
   - Email
   - GitHub
   - Google
3. Vérifier votre email si nécessaire

### Étape 2 : Préparer les Fichiers

Assurez-vous que votre dossier `py_backend` contient :
- ✅ `Dockerfile`
- ✅ `requirements.txt`
- ✅ `main.py`
- ✅ Tous les autres fichiers Python
