┌───────────────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 1 : INSTALLER KOYEB CLI                                                │
└───────────────────────────────────────────────────────────────────────────────┘

  Windows (Git Bash/WSL) :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ curl -fsSL https://raw.githubusercontent.com/koyeb/koyeb-cli/master/... │
  │ export PATH=$HOME/.koyeb/bin:$PATH                                      │
  └─────────────────────────────────────────────────────────────────────────┘

  Vérifier :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb version                                                           │
  └─────────────────────────────────────────────────────────────────────────┘


┌───────────────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 2 : SE CONNECTER                                                       │
└───────────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb login                                                             │
  └─────────────────────────────────────────────────────────────────────────┘

  → Suivre les instructions dans le navigateur


┌───────────────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 3 : VÉRIFIER (OPTIONNEL)                                               │
└───────────────────────────────────────────────────────────────────────────────┘

  Windows PowerShell :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ .\check-koyeb-ready.ps1                                                 │
  └─────────────────────────────────────────────────────────────────────────┘

  Linux/Mac Bash :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ chmod +x check-koyeb-ready.sh                                           │
  │ ./check-koyeb-ready.sh                                                  │
  └─────────────────────────────────────────────────────────────────────────┘


┌───────────────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 4 : DÉPLOYER                                                           │
└───────────────────────────────────────────────────────────────────────────────┘

  Windows PowerShell :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ .\deploy-koyeb.ps1 dev                                                  │
  └─────────────────────────────────────────────────────────────────────────┘

  Linux/Mac Bash :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ chmod +x deploy-koyeb.sh                                                │
  │ ./deploy-koyeb.sh dev                                                   │
  └─────────────────────────────────────────────────────────────────────────┘

  OU Commande Manuelle :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ cd py_backend                                                           │
  │ koyeb deploy . clara-backend/clara-service \                            │
  │   --archive-builder docker \                                            │
  │   --type WEB \                                                          │
  │   --ports 5000:http \                                                   │
  │   --routes /:5000 \                                                     │
  │   --regions fra \                                                       │
  │   --instance-type nano                                                  │
  └─────────────────────────────────────────────────────────────────────────┘

  └─────────────────────────────────────────────────────────────────────────┘

  Voir les logs :

  Tester l'API :

┌───────────────────────────────────────────────────────────────────────────────┐
│ 🌐 ACCÉDER À L'API                                                           │
└───────────────────────────────────────────────────────────────────────────────┘

  URL de base :
  https://clara-backend-[your-org].koyeb.app

  Endpoints principaux :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ GET  /              → Root endpoint                                     │
  │ GET  /health        → Health check                                      │
  │ GET  /docs          → Documentation Swagger UI                          │
  │ GET  /redoc         → Documentation ReDoc                               │
  │                                                                         │
  │ POST /notebooks     → Créer un notebook LightRAG                        │
  │ GET  /notebooks     → Lister les notebooks                              │
  │ POST /notebooks/{id}/documents → Upload documents                       │
  │ POST /notebooks/{id}/query     → Interroger un notebook                 │
  │                                                                         │
  │ POST /pandas/analyze → Analyse de données avec Pandas                   │
  │ POST /word/export    → Export Word                                      │
  │ POST /speech-to-text → Transcription audio                              │
  │ POST /text-to-speech → Synthèse vocale                                  │
  └─────────────────────────────────────────────────────────────────────────┘


┌───────────────────────────────────────────────────────────────────────────────┐
│ 🔧 COMMANDES UTILES                                                          │
└───────────────────────────────────────────────────────────────────────────────┘

  Voir les logs en temps réel :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb service logs clara-backend/clara-service --follow                 │
  └─────────────────────────────────────────────────────────────────────────┘

  Voir les logs de build :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb service logs clara-backend/clara-service --type build             │
  └─────────────────────────────────────────────────────────────────────────┘

  Mettre à jour le service :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ cd py_backend                                                           │
  │ koyeb deploy . clara-backend/clara-service                              │
  └─────────────────────────────────────────────────────────────────────────┘

  Redémarrer le service :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb service redeploy clara-backend/clara-service                      │
  └─────────────────────────────────────────────────────────────────────────┘

  Scaler le service :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb service update clara-backend/clara-service \                      │
  │   --min-scale 2 --max-scale 4                                           │
  └─────────────────────────────────────────────────────────────────────────┘

  Changer le type d'instance :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb service update clara-backend/clara-service \                      │
  │   --instance-type small                                                 │
  └─────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb service logs clara-backend/clara-service --type build             │
  │ koyeb service logs clara-backend/clara-service                          │
  └─────────────────────────────────────────────────────────────────────────┘

  Erreur de mémoire :
  ┌─────────────────────────────────────────────────────────────────────────┐
  │ koyeb service update clara-backend/clara-service \                      │
  │   --instance-type small                                                 │
  └─────────────────────────────────────────────────────────────────────────┘

  Timeout lors du build :
  → Attendre quelques minutes (installation de spaCy, etc.)

  → Consulter GUIDE_DEPLOIEMENT_KOYEB.md (section Dépannage)
