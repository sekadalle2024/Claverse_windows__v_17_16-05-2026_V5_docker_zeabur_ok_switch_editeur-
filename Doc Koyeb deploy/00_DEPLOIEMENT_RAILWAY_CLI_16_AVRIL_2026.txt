🎯 OBJECTIF
═══════════════════════════════════════════════════════════════════════════════

Déployer le backend Python sur Railway Hobby ($5/mois) via CLI
pour éviter le sleep mode de Koyeb Free et garantir une disponibilité 24/7.


📋 PRÉREQUIS
═══════════════════════════════════════════════════════════════════════════════

✅ Node.js installé (pour Railway CLI)
✅ Compte Railway créé (https://railway.app)
✅ Backend dans py_backend/ avec Dockerfile
✅ Carte bancaire pour Railway Hobby ($5/mois)


⚡ DÉMARRAGE ULTRA-RAPIDE (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

1️⃣ INSTALLER RAILWAY CLI
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ npm install -g @railway/cli                                            │
   │ railway --version                                                       │
   └─────────────────────────────────────────────────────────────────────────┘

2️⃣ SE CONNECTER À RAILWAY
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ railway login                                                           │
   └─────────────────────────────────────────────────────────────────────────┘

3️⃣ DÉPLOYER LE BACKEND (AUTOMATIQUE)
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ cd py_backend                                                           │
   │ railway init                                                            │
   │ railway up                                                              │
   └─────────────────────────────────────────────────────────────────────────┘

4️⃣ RÉCUPÉRER L'URL
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ railway domain                                                          │
   │ # Exemple: clara-backend-production.up.railway.app                      │
   └─────────────────────────────────────────────────────────────────────────┘

5️⃣ ⚠️ VÉRIFICATION CRITIQUE
   • Aller sur: https://railway.app/dashboard
   • Sélectionner votre projet
   • Settings → Sleep Settings
   • VÉRIFIER: "App Sleeping" est DÉSACTIVÉ (OFF)

6️⃣ METTRE À JOUR LE FRONTEND
   • Fichier: src/services/claraApiService.ts
   • Remplacer: hkj0631c.rpcl.app
   • Par: clara-backend-production.up.railway.app


🤖 DÉPLOIEMENT AUTOMATISÉ (RECOMMANDÉ)
═══════════════════════════════════════════════════════════════════════════════

Utilisez le script PowerShell pour automatiser tout le processus :

   ┌─────────────────────────────────────────────────────────────────────────┐
   │ # Depuis la racine du projet                                            │
   │ .\Doc Koyeb deploy\Scripts\deploy-railway.ps1                          │
   │                                                                         │
   │ # Ou avec options                                                       │
   │ .\Doc Koyeb deploy\Scripts\deploy-railway.ps1 -ProjectName "mon-backend" │
   └─────────────────────────────────────────────────────────────────────────┘

Le script fait automatiquement :
   ✅ Vérification des prérequis
   ✅ Connexion à Railway
   ✅ Initialisation du projet
   ✅ Configuration des variables d'environnement
   ✅ Déploiement du backend
   ✅ Tests post-déploiement
   ✅ Affichage de l'URL
