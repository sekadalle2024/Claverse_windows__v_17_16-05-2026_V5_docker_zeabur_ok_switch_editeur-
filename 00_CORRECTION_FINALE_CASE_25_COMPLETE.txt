PROBLÈME RÉSOLU
───────────────────────────────────────────────────────────────────────────
Les tableaux 2-6 s'affichaient en une seule ligne sans structure:
| Intitule | |──────────────| | Défaut de sécurisation...

CAUSE
───────────────────────────────────────────────────────────────────────────
Le séparateur utilisait des caractères Unicode `─` (U+2500) au lieu de
tirets simples `-` (U+002D) requis par la spécification Markdown.

SOLUTION APPLIQUÉE
───────────────────────────────────────────────────────────────────────────
Remplacer `'─'` par `'-'` dans le séparateur du tableau Markdown.

FICHIER MODIFIÉ
───────────────────────────────────────────────────────────────────────────
src/services/claraApiService.ts
→ Fonction convertHeaderTableToMarkdown() (ligne ~665)

CHANGEMENT
───────────────────────────────────────────────────────────────────────────
AVANT: `|${'─'.repeat(headerText.length + 6)}|\n`  ❌ Unicode
APRÈS:  `|${'-'.repeat(headerText.length + 6)}|\n`  ✅ Tirets simples

CODE FINAL
───────────────────────────────────────────────────────────────────────────
```typescript
// Tables 2+ (index > 0) : Format 1 colonne avec en-tête en gras
if (entries.length === 1) {
  const [key, value] = entries[0];
  const headerText = key.charAt(0).toUpperCase() + key.slice(1);
  let cellValue = String(value)
    .replace(/\\n/g, '\n')
    .replace(/\n/g, '\n\n')
    .trim();
  
  // Tableau Markdown 1 colonne avec TIRETS SIMPLES
  let md = `\n| **${headerText}** |\n`;
  md += `|${'-'.repeat(headerText.length + 6)}|\n`;  // ✅ TIRETS SIMPLES
  md += `| ${cellValue.replace(/\n\n/g, '<br><br>')} |\n\n`;
  
  return md;
}
```
