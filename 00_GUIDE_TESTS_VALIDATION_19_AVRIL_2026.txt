🔍 DIAGNOSTIC EN CAS DE PROBLÈME

Si les tests échouent, vérifier dans cet ordre :

1. CACHE NAVIGATEUR
   ─────────────────
   - Faire Ctrl + Shift + R pour vider le cache
   - Ou ouvrir en navigation privée

2. ZEABUR LOGS
   ────────────
   - Aller sur https://zeabur.com/
   - Ouvrir le projet backend
   - Consulter les logs de déploiement
   - Vérifier que l'endpoint /etats-financiers/process-excel existe

3. CONSOLE NAVIGATEUR
   ───────────────────
   - Ouvrir F12
   - Onglet Console
   - Chercher les erreurs en rouge
   - Noter le message d'erreur exact

4. NETWORK TAB
   ───────────
   - F12 → Onglet Network
   - Refaire l'upload
   - Cliquer sur la requête qui échoue
   - Vérifier :
     * Status Code (doit être 200, pas 404 ou 500)
     * Response Headers (CORS)
     * Response Body (message d'erreur)
