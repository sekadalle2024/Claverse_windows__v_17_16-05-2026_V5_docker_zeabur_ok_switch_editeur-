🎯 PROBLÈME IDENTIFIÉ
─────────────────────────────────────────────────────────────────────────────

Le webhook retourne :
  { data: {...} }  ← OBJET

Mais claraApiService.ts attend :
  [{ data: {...} }]  ← ARRAY

La condition dans claraApiService.ts (ligne 1233) est :
  if (Array.isArray(result) && result.length > 0)

Cette condition échoue car result est un OBJET, pas un ARRAY !


🔍 CAUSE RACINE
─────────────────────────────────────────────────────────────────────────────

Dans n8n, quand le node Code retourne :
  return [{ data: finalData }];

n8n transforme automatiquement cette structure et le webhook envoie :
  { data: {...} }  ← Le premier élément de l'array

Pour que le webhook envoie un ARRAY, il faut que le node Code retourne :
  return [{ json: [{ data: finalData }] }];

Ainsi, n8n transforme et le webhook envoie :
  [{ data: {...} }]  ← ARRAY avec un élément


✅ SOLUTION
─────────────────────────────────────────────────────────────────────────────
