  ✅ CORRECTION EXPORT TEMPLATE EXCEL - TOUTES LES TABLES
  Date: 09 Mai 2026
  Problème résolu: Export de toutes les tables dans la div (pas seulement la table active)
═══════════════════════════════════════════════════════════════════════════════

📋 PROBLÈME IDENTIFIÉ
═══════════════════════════════════════════════════════════════════════════════

Lorsque l'utilisateur exécute l'action "Exporter template Excel" depuis le menu contextuel,
seule la table active est exportée en Excel sur le desktop.

Les autres tables présentes dans la même div ne sont PAS exportées.

🔍 CAUSE RACINE
═══════════════════════════════════════════════════════════════════════════════

La fonction exportTemplate() utilisait une approche complexe:
1. Chercher le conteneur parent de la table active
2. Chercher les tables dans ce conteneur

❌ PROBLÈME: Le conteneur n'était pas trouvé correctement car:
   - Les classes CSS complètes sont: "prose prose-base dark:prose-invert max-w-none"
   - Le sélecteur closest('div.prose.prose-base.dark\\:prose-invert.max-w-none') 
     ne fonctionnait pas de manière fiable

✅ SOLUTION APPLIQUÉE
═══════════════════════════════════════════════════════════════════════════════

Utiliser le MÊME sélecteur que exportSyntheseCAC() qui fonctionne parfaitement:

  document.querySelectorAll('div.prose table')

Ce sélecteur:
✅ Trouve TOUTES les tables dans les divs avec la classe "prose"
✅ Fonctionne de manière fiable et testée
✅ Est déjà utilisé avec succès dans exportSyntheseCAC()

🔧 CHANGEMENTS TECHNIQUES
═══════════════════════════════════════════════════════════════════════════════

AVANT (approche complexe qui ne fonctionnait pas):
─────────────────────────────────────────────────────────────────────────────
let tables = [];

if (this.targetTable) {
  const container = this.targetTable.closest('div.prose.prose-base...')
    || this.targetTable.closest('.chat-message')
    || ...;
  
  if (container) {
    tables = Array.from(container.querySelectorAll('table'));
  }
}

if (tables.length === 0) {
  tables = Array.from(document.querySelectorAll('div.prose table'));
}
─────────────────────────────────────────────────────────────────────────────

APRÈS (approche simple et efficace):
─────────────────────────────────────────────────────────────────────────────
// Utiliser directement le sélecteur global 'div.prose table'
// C'est le même sélecteur que exportSyntheseCAC qui fonctionne correctement
const tables = Array.from(document.querySelectorAll('div.prose table'));

console.log(`✅ [Export Template] Sélecteur utilisé: div.prose table`);
console.log(`🔍 [Export Template] ${tables.length} table(s) trouvée(s)`);
─────────────────────────────────────────────────────────────────────────────
