  ✅ CORRECTION EXPORT TEMPLATE EXCEL - TOUTES LES TABLES
═══════════════════════════════════════════════════════════════════════════════

📅 Date: 09 Mai 2026
🎯 Objectif: Corriger l'export template Excel pour exporter TOUTES les tables de la div
📝 Statut: ✅ CORRECTION APPLIQUÉE

═══════════════════════════════════════════════════════════════════════════════
  📋 PROBLÈME IDENTIFIÉ
═══════════════════════════════════════════════════════════════════════════════

❌ SYMPTÔME:
   L'action "Exporter template Excel" n'exportait que la table active
   Les autres tables dans la div n'étaient pas exportées

🔍 CAUSE RACINE:
   La fonction exportTemplate() utilisait findRelatedTables() qui avait une
   logique de détection limitée et ne trouvait pas toutes les tables

═══════════════════════════════════════════════════════════════════════════════
  💡 SOLUTION APPLIQUÉE
═══════════════════════════════════════════════════════════════════════════════

✅ APPROCHE:
   Utiliser le même sélecteur CSS que la fonction exportSyntheseCAC()
   qui fonctionne parfaitement pour détecter toutes les tables

🔧 SÉLECTEUR CSS UTILISÉ:
   document.querySelectorAll('div.prose table')

   Ce sélecteur détecte TOUTES les tables dans les divs avec la classe "prose"
   qui est la classe standard des messages dans Claraverse

📝 CODE MODIFIÉ:
   Fichier: public/menu.js
   Fonction: exportTemplate()

   AVANT:
   -------
   let tables = this.findRelatedTables();

   APRÈS:
   ------
   const allTables = Array.from(document.querySelectorAll('div.prose table'));
   let tables = allTables.length > 0 ? allTables : this.findRelatedTables();

═══════════════════════════════════════════════════════════════════════════════
  🎯 AVANTAGES DE LA SOLUTION
═══════════════════════════════════════════════════════════════════════════════

✅ Détection universelle de toutes les tables dans la div
✅ Même approche que exportSyntheseCAC (cohérence du code)
✅ Fallback sur findRelatedTables() si le sélecteur global ne trouve rien
✅ Logs de diagnostic pour faciliter le débogage
✅ Solution simple et robuste
