═══════════════════════════════════════════════════════════════════════════
🚀 DÉPLOIEMENT EN 3 ÉTAPES
═══════════════════════════════════════════════════════════════════════════

┌───────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 1: PUSH VERS GITHUB (2 minutes)                                │
├───────────────────────────────────────────────────────────────────────┤
│                                                                       │
│   Exécuter le script automatique:                                    │
│   .\Doc zeabur docker\Scripts\push-to-github-zeabur.ps1              │
│                                                                       │
│   OU manuellement:                                                   │
│   git add docker-compose.yml Dockerfile.frontend                     │
│   git add py_backend/Dockerfile                                      │
│   git commit -m "feat: Docker Compose pour Zeabur"                   │
│   git push origin main                                                │
│                                                                       │
└───────────────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 2: CONFIGURATION ZEABUR (5 minutes)                            │
├───────────────────────────────────────────────────────────────────────┤
│                                                                       │
│   A. Créer un projet                                                 │
│      • Aller sur https://zeabur.com/dashboard                        │
│      • New Project → "claraverse-production"                         │
│                                                                       │
│   B. Ajouter le service Backend                                      │
│      • Add Service → Git Repository                                  │
│      • Choisir votre repo                                            │
│      • Build Method: Docker Compose                                  │
│      • Service: backend                                              │
│      • Port: 8080                                                    │
│      • Domain: pybackend.zeabur.app                                  │
│                                                                       │
│   C. Ajouter le service Frontend                                     │
│      • Add Service → Git Repository                                  │
│      • Même repo                                                     │
│      • Build Method: Docker Compose                                  │
│      • Service: frontend                                             │
│      • Port: 80                                                      │
│      • Domain: prclaravi.zeabur.app                                  │
│                                                                       │
└───────────────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 3: DÉPLOIEMENT (10-15 minutes)                                 │
├───────────────────────────────────────────────────────────────────────┤
│                                                                       │
│   • Cliquer "Deploy" sur chaque service                              │
│   • Zeabur va:                                                       │
│     1. Cloner le repository GitHub                                   │
│     2. Lire docker-compose.yml                                       │
│     3. Builder les images Docker                                     │
│     4. Déployer les conteneurs                                       │
│                                                                       │
│   • Suivre les logs de build dans Zeabur                            │
│   • Attendre que les services soient "Running"                       │
│                                                                       │
└───────────────────────────────────────────────────────────────────────┘

🔄 WORKFLOW DE DÉVELOPPEMENT CONTINU
═══════════════════════════════════════════════════════════════════════════

1. Développer localement (SANS Docker):
   • Frontend: npm run dev (localhost:5173)
   • Backend: python main.py (localhost:5000)

2. Tester localement

3. Commit et push:
   git add .
   git commit -m "feat: nouvelle fonctionnalité"
   git push origin main

4. Zeabur redéploie automatiquement (10-15 minutes)

5. Vérifier en production:
   https://prclaravi.zeabur.app
