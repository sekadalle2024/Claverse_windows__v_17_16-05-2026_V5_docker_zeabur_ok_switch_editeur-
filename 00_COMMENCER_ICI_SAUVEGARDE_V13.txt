  EN CAS DE PROBLÈME
═══════════════════════════════════════════════════════════════════════════════

Erreur: "HTTP 408 Timeout"
→ Utiliser la Méthode 2 (commits multiples)
→ Ou utiliser GitHub Desktop

Erreur: "Connection reset"
→ Vérifier la connexion Internet
→ Attendre quelques minutes
→ Réessayer

Erreur: "cannot lock ref"
→ git fetch origin master
→ git merge origin/master --allow-unrelated-histories
→ git push -u origin master

Projet trop volumineux (> 100 MB)
→ Utiliser la Méthode 2 (commits multiples)
→ Lire: Doc_Github_Issue/SOLUTION_PROJET_140MB_16_AVRIL_2026.md
