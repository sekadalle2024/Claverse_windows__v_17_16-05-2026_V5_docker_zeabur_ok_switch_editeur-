═══════════════════════════════════════════════════════════════
  POURQUOI CE DIAGNOSTIC ?
═══════════════════════════════════════════════════════════════

Le système d'export du rapport consolidé CAC ne détecte pas les
tables dans le chat. Ce diagnostic va identifier:

✅ Combien de tables sont présentes
✅ Quelles classes CSS elles utilisent
✅ Quel sélecteur CSS fonctionne le mieux

Une fois ces informations obtenues, je pourrai corriger le code
dans menu.js pour détecter correctement vos tables.
