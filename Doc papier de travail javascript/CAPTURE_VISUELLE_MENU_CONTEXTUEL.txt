═══════════════════════════════════════════════════════════════════════════════
  🔄 CYCLE DE VIE COMPLET
═══════════════════════════════════════════════════════════════════════════════

1. CRÉATION
   Clic droit → Papier de travail → Ajouter Schéma de calcul
   OU
   Ctrl+Shift+K

2. MODIFICATION
   Cliquer dans les cellules du schéma pour les éditer
   Les modifications sont sauvegardées automatiquement

3. ACTUALISATION
   Clic droit → Papier de travail → Actualiser Schéma de calcul
   Le schéma est régénéré (contenu réinitialisé)

4. SUPPRESSION
   Clic droit → Papier de travail → Supprimer Schéma de calcul
   Confirmation → Le schéma disparaît
