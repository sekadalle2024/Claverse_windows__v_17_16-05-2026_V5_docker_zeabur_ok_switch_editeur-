  DÉPANNAGE RAPIDE
═══════════════════════════════════════════════════════════════════════════════

❌ PROBLÈME : "Le module Schéma de Calcul n'est pas chargé"

SOLUTION :
1. Ouvrir index.html
2. Vérifier la présence de :
   <script src="/papier-travail-schema-calcul.js"></script>
3. Vérifier que ce script est chargé AVANT menu.js

❌ PROBLÈME : Le menu ne s'affiche pas

SOLUTION :
1. Ouvrir la console (F12)
2. Vérifier les erreurs JavaScript
3. Vérifier que menu.js est bien chargé
4. Vérifier que initContextualMenu() est appelé

❌ PROBLÈME : Le schéma n'apparaît pas

SOLUTION :
1. Vérifier que la table est modelisée (colonnes Conclusion/Assertion/CTR)
2. Vérifier que la table a une "Nature de test" dans la table 2
3. Ouvrir la console et chercher les logs "[Schéma Calcul]"
