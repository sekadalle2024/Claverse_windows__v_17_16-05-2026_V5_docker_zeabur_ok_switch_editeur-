📊 FONCTIONNALITÉ : Totalisation automatique des colonnes numériques

═══════════════════════════════════════════════════════════════════════════════
  🎯 OBJECTIF
═══════════════════════════════════════════════════════════════════════════════

Ajouter une ligne de totalisation en bas des tables de test avec:
- Détection automatique des colonnes numériques
- Calcul automatique des totaux
- Actualisation en temps réel
- Persistance après actualisation

═══════════════════════════════════════════════════════════════════════════════
  📋 PRÉREQUIS
═══════════════════════════════════════════════════════════════════════════════

1. ✅ Fichier créé: public/papier-travail-totalisation.js
2. ✅ Méthodes ajoutées dans public/menu.js:
   - ajouterTotalisation()
   - actualiserTotalisation()
   - supprimerTotalisation()
3. ✅ Raccourci clavier: Ctrl+Shift+T
4. ✅ Menu contextuel: Section "Papier de travail"

═══════════════════════════════════════════════════════════════════════════════
  🔧 ÉTAPE 1 : CHARGER LE MODULE
═══════════════════════════════════════════════════════════════════════════════

Ajouter dans index.html (avant la fermeture de </body>):

<script src="/papier-travail-totalisation.js"></script>

Position recommandée:
- Après papier-travail-schema-calcul.js
- Après papier-travail-cross-ref-horizontale.js
- Avant menu.js

═══════════════════════════════════════════════════════════════════════════════
  🧪 ÉTAPE 2 : TEST BASIQUE
═══════════════════════════════════════════════════════════════════════════════

1. Ouvrir l'application ClaraVerse
2. Créer ou ouvrir une table de test avec des colonnes numériques
3. Clic droit sur la table
4. Menu "Papier de travail" → "Ajouter Totalisation"
5. Vérifier qu'une ligne "Totalisation" apparaît en bas

RÉSULTAT ATTENDU:
✅ Ligne de totalisation ajoutée
✅ Totaux calculés pour les colonnes numériques
✅ Fond jaune clair
✅ Texte en gras

═══════════════════════════════════════════════════════════════════════════════
  🧪 ÉTAPE 3 : TEST ACTUALISATION AUTOMATIQUE
═══════════════════════════════════════════════════════════════════════════════

3. Vérifier que le total se met à jour automatiquement

RÉSULTAT ATTENDU:
✅ Total recalculé automatiquement
✅ Pas besoin de cliquer sur "Actualiser"

  🐛 DÉPANNAGE
═══════════════════════════════════════════════════════════════════════════════

PROBLÈME: Module non chargé
SOLUTION: Vérifier dans la console:
  window.PapierTravailTotalisation
  → Doit retourner un objet avec les méthodes

PROBLÈME: Aucune colonne numérique détectée
SOLUTION: Vérifier dans la console:
  totalisationCommands.showStorage()
  → Voir les colonnes détectées

PROBLÈME: Totaux incorrects
SOLUTION: Actualiser manuellement:
  Menu → "Actualiser Totalisation"

PROBLÈME: Totalisation disparaît après F5
SOLUTION: Vérifier localStorage:
  totalisationCommands.showStorage()
  → Doit contenir les données
