✅ Positionnement au-dessus de la table principale
✅ Sauvegarde persistante avec localStorage

🚀 DÉMARRAGE RAPIDE
═══════════════════════════════════════════════════════════════════════════════

1. VÉRIFIER L'INTÉGRATION
   - Les scripts sont déjà dans public/
   - Le service TypeScript est configuré
   - Le module se charge automatiquement

2. UTILISER LE SCHÉMA DE CALCUL
   
   A) AUTOMATIQUE (recommandé):
      - Le module détecte automatiquement les tables
      - Génère le schéma selon la "Nature de test"
      - Positionne au-dessus de la table principale
   
   B) MANUEL (via menu contextuel):
      - Clic droit sur une table
      - Menu "Papier de travail"
      - "📐 Ajouter Schéma de calcul" (Ctrl+Shift+S)

3. ÉDITER LE SCHÉMA
   - Cliquer sur une cellule du schéma
   - Modifier le texte
   - Appuyer sur Entrée ou cliquer ailleurs
   - Sauvegarde automatique dans localStorage

📊 NATURES DE TEST SUPPORTÉES
═══════════════════════════════════════════════════════════════════════════════

✅ Validation          → (A), (B), (C) = (A) + (B), (D), (E) = (C) - (D)
✅ Mouvement           → (A), (B), (C), (D) = (A) + (B) - (C), (E), (F) = (D) - (E)
✅ Rapprochement       → (A), (B), (C) = (A) - (B)
✅ Séparation          → (A), (B), (C) = (A) - (B)
✅ Estimation          → (A), (B), (C) = (A) * (B), (D), (E) = (C) - (D)
✅ Revue analytique    → (A), (B), (C) = (A) - (B)
✅ Cadrage TVA         → (A), (B) = (A) * 18%, (C), (D), (E), (F) = (B) - (C) - (D) - (E)
✅ Cotisations sociales → (A), (B), (C), (D)
✅ Modélisation        → Détection automatique des variables (X), (Y), (Z)...
✅ Vierge              → Aucune colonne (pour personnalisation)
