QUESTION :
"L'action ne figure pas au menu contextuel dans la section papier de travail ?"

RÉPONSE :
✅ SI, les actions SONT BIEN intégrées dans le menu contextuel !

Les 3 actions du schéma de calcul sont présentes dans la section 
"📁 Papier de travail" du menu contextuel :

1. 📐 Ajouter Schéma de calcul (Ctrl+Shift+K)
2. 🔄 Actualiser Schéma de calcul
3. 🗑️ Supprimer Schéma de calcul

═══════════════════════════════════════════════════════════════════════════════
  🧪 COMMENT VÉRIFIER
═══════════════════════════════════════════════════════════════════════════════

1. Ouvrir l'application ClaraVerse
2. Créer ou ouvrir une table modelisée
3. Clic droit sur la table
4. Chercher la section "📁 Papier de travail"
5. Les 3 actions doivent être visibles

Si vous ne voyez pas les actions, vérifiez :
- Que menu.js est bien chargé
- Qu'il n'y a pas d'erreurs JavaScript dans la console (F12)
- Que vous faites bien un clic droit sur une TABLE (pas ailleurs)

═══════════════════════════════════════════════════════════════════════════════

1. AJOUTER SCHÉMA DE CALCUL
   ✓ Vérifie la table active
   ✓ Vérifie que SchemaCalculManager est chargé
   ✓ Trouve le conteneur parent (div.prose)
   ✓ Déclenche la création du schéma
   ✓ Affiche notification de succès
   ✓ Gestion d'erreurs complète

2. ACTUALISER SCHÉMA DE CALCUL
   ✓ Supprime le schéma existant
   ✓ Retire la table de la liste des tables traitées
   ✓ Retraite la div pour régénérer le schéma
   ✓ Notification de succès

3. SUPPRIMER SCHÉMA DE CALCUL
   ✓ Trouve le schéma associé via data-for-table
   ✓ Supprime du DOM
   ✓ Supprime du localStorage
   ✓ Confirmation utilisateur
   ✓ Notification de succès

4. RACCOURCI CLAVIER
   ✓ Ctrl+Shift+K pour ajouter un schéma
