│                      📊 NATURES DE TEST SUPPORTÉES                          │
└─────────────────────────────────────────────────────────────────────────────┘

1. Validation          → (A), (B), (C) = (A)+(B)
2. Mouvement           → (A), (B), (C), (D) = (A+B-C), (E), (F) = (D)-(E)
3. Rapprochement       → (A), (B), (C) = (A) - (B)
4. Séparation          → (A), (B), (C) = (A) - (B)
5. Estimation          → A, B, C = A*B, D, E = C - D
6. Revue analytique    → (A), (B), (C) = (A) - (B)
7. Cadrage TVA         → (A), (B) = (A)*18%, (C), (D), (E), (F) = (B)-(C)-(D)-(E)
8. Cotisations         → (A), (B), (C) = (A) - (B)
9. Fallback Regex      → Extraction automatique depuis table 4b
