🔍 Diagnostic Collecte Tables Claraverse - Outil créé

## 🎯 Problème Identifié

HTTP REQUEST → n8n fonctionne ✅
Claraverse → n8n envoie des données vides ❌

Hypothèse : Les tables ne sont pas collectées correctement dans
Claraverse par la fonction collectCriteriaTables()

## 🧪 Solution

Création d'un outil de diagnostic interactif pour identifier
pourquoi les tables ne sont pas collectées.

## 🔍 Fonctionnalités de l'Outil

1. Analyse du DOM
   - Compte les divs "prose"
   - Compte les tables
   - Vérifie les en-têtes requis
   - Cherche les mots-clés

2. Test de collectCriteriaTables()
   - Simule la fonction
   - Affiche les tables collectées
   - Montre le HTML généré

3. Statistiques en Temps Réel
   - Divs Prose
   - Tables Totales
   - Avec En-têtes Requis
   - Correspondantes

4. Logs Détaillés
   - Analyse div par div
   - Détection des problèmes
   - Recommandations de solutions

## 🚀 Utilisation

```bash
# Démarrer Claraverse
npm run dev

# Créer des tables de test
# (voir ACTION_IMMEDIATE_COLLECTE_TABLES.txt)

# Ouvrir l'outil
http://localhost:5173/diagnostic-collecte-tables-claraverse.html

# Cliquer sur "Analyser le DOM"
```
