🔧 ARCHITECTURE DU WORKFLOW V3
═══════════════════════════════════════════════════════════════════════════════

Le workflow contient 12 nodes :

1. Cross References Form (Form Trigger)
   └─> Formulaire d'upload avec 6 champs

2. Rechercher Dossier CAC (Code)
   └─> Utilise $http.request() pour chercher "Dossier CAC"

3. Dossier CAC existe? (IF)
   ├─> OUI : Continue vers recherche dossier client
   └─> NON : Crée le dossier CAC

4. Créer Dossier CAC (Code)
   └─> Utilise $http.request() pour créer le dossier

5. Préparer Recherche Client (Code)
   └─> Prépare les données pour la recherche

6. Rechercher Dossier Client (Code)
   └─> Utilise $http.request() pour chercher le dossier client

7. Dossier Client existe? (IF)
   ├─> OUI : Continue vers recherche cycle
   └─> NON : Crée le dossier client + cycles

8. Créer Dossier Client (Code)
   └─> Utilise $http.request() pour créer le dossier client

9. Créer Cycles (Code)
   └─> Utilise une boucle pour créer les 10 cycles

10. Préparer Recherche Cycle (Code)
    └─> Prépare les données pour la recherche du cycle

11. Rechercher Cycle (Code)
    └─> Utilise $http.request() pour chercher le cycle

12. Upload Fichier (Code)
    └─> Utilise $http.request() avec multipart pour uploader

🔑 CONFIGURATION OAUTH2
═══════════════════════════════════════════════════════════════════════════════

1. Créer un credential Google OAuth2 dans n8n :
   
   Settings > Credentials > New Credential > Google OAuth2 API
   
   Client ID: [VOTRE_GOOGLE_CLIENT_ID]
   Client Secret: [VOTRE_GOOGLE_CLIENT_SECRET]
   OAuth Redirect URL: https://hkj0631c.rpcl.app/rest/oauth2-credential/callback
   
   Scopes:
   - https://www.googleapis.com/auth/drive
   - https://www.googleapis.com/auth/drive.file

2. Connecter le compte Google

3. Assigner le credential aux nodes Code :
   - Rechercher Dossier CAC
   - Créer Dossier CAC
   - Rechercher Dossier Client
   - Créer Dossier Client
   - Créer Cycles
   - Rechercher Cycle
   - Upload Fichier

💡 COMMENT ÇA FONCTIONNE
═══════════════════════════════════════════════════════════════════════════════

Chaque node Code utilise :

1. await this.getCredentials('googleOAuth2Api')
   → Récupère le token OAuth2

2. credentials.oauthTokenData.access_token
   → Extrait le token d'accès

3. await $http.request({ ... })
   → Fait la requête HTTP à l'API Google Drive

4. headers: { 'Authorization': `Bearer ${accessToken}` }
   → Authentifie la requête
