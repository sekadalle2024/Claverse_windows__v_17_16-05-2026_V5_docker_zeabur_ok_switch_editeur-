🎯 DÉMARRAGE RAPIDE (5 ÉTAPES)
═══════════════════════════════════════════════════════════════════════════════

ÉTAPE 1 : CRÉER LE CREDENTIAL OAUTH2
────────────────────────────────────────────────────────────────────────────────
1. Ouvrir n8n
2. Settings > Credentials > New Credential
3. Choisir "Google OAuth2 API"
4. Remplir :
   Client ID: [VOTRE_GOOGLE_CLIENT_ID]
   Client Secret: [VOTRE_GOOGLE_CLIENT_SECRET]
   Scopes: 
   - https://www.googleapis.com/auth/drive
   - https://www.googleapis.com/auth/drive.file
5. Connect my account > Autoriser
6. Save

📖 Détails : Voir GUIDE_CONFIGURATION_OAUTH2_GOOGLE_DRIVE.md

ÉTAPE 2 : IMPORTER LE WORKFLOW
────────────────────────────────────────────────────────────────────────────────
1. Ouvrir n8n
2. Cliquer sur "+" (nouveau workflow)
3. Menu (⋮) > Import from File
4. Sélectionner : WORKFLOW_N8N_XREF_HTTP_SOLUTION.json
5. Le workflow s'ouvre

ÉTAPE 3 : CONFIGURER LES CREDENTIALS
────────────────────────────────────────────────────────────────────────────────
Pour CHAQUE node HTTP Request (7 nodes au total) :
1. Cliquer sur le node
2. Authentication > OAuth2
3. Credential to connect with > Sélectionner le credential créé
4. Fermer le node

Les 7 nodes à configurer :
✓ Rechercher Dossier CAC
✓ Créer Dossier CAC
✓ Rechercher Dossier Client
✓ Créer Dossier Client
✓ Créer Cycles
✓ Rechercher Cycle
✓ Upload Fichier

ÉTAPE 4 : ACTIVER LE WORKFLOW
────────────────────────────────────────────────────────────────────────────────
1. Cliquer sur "Active" en haut à droite
2. Le workflow est maintenant actif

ÉTAPE 5 : TESTER
────────────────────────────────────────────────────────────────────────────────
1. Cliquer sur le node "Cross References Form"
2. Copier l'URL du formulaire
3. Ouvrir l'URL dans un navigateur
4. Remplir le formulaire :
   - Client: Orange
   - Exercice: 2025
   - Cycle: Trésorerie
   - Cross référence: I1
   - Document: Lead sheet trésorerie
   - Fichier: Sélectionner un fichier test
5. Submit
6. Vérifier dans Google Drive

🔧 DÉPANNAGE RAPIDE
═══════════════════════════════════════════════════════════════════════════════

ERREUR 401 Unauthorized
→ Le credential OAuth2 n'est pas configuré
→ Solution : Configurer le credential dans chaque node HTTP Request

ERREUR 403 Forbidden
→ Les scopes sont incorrects
→ Solution : Vérifier les 2 scopes dans le credential

Le fichier n'est pas uploadé
→ Problème avec les binary data
→ Solution : Vérifier le node "Préparer Upload"

Les dossiers ne sont pas créés
→ Permissions Google Drive
→ Solution : Vérifier les permissions du compte Google
