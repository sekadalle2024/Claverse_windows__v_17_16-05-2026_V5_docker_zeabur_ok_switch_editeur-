📋 ÉTAPE 1: CRÉER UNE TABLE X-REF DANS LE CHAT
───────────────────────────────────────────────

Copiez et collez ce tableau dans le chat Claraverse:

| No | Cross references | Document |
|----|------------------|----------|
| 1  | [I1]             | Lead sheet trésorerie |
| 2  | [I20]            | Proces verbal inventaire de caisse |
| 3  | [I150]           | Balances generle |


📤 ÉTAPE 2: TESTER L'IMPORT DE DOCUMENT
────────────────────────────────────────

1. Cliquez droit sur la table
2. Menu contextuel → "Papier de travail" → "📤 Importer X-Ref documentaire"
3. OU utilisez le raccourci: Ctrl+Shift+X

Le formulaire n8n s'ouvre. Remplissez:
- Client: Orange
- Exercice: 2025
- Cycle comptable: Trésorerie
- Cross référence: I20
- Document: Proces verbal inventaire de caisse
- Fichier: (sélectionnez un PDF ou Excel)

4. Cliquez "Submit"

✅ Le document est uploadé dans:
   Dossier CAC/Orange-2025/Trésorerie/[I20]-Proces verbal inventaire de caisse


📋 ÉTAPE 3: AFFICHER LES DOCUMENTS
───────────────────────────────────

1. Cliquez droit sur la table
2. Menu contextuel → "Papier de travail" → "📋 Afficher X-Ref documentaire"

Une barre latérale s'ouvre à droite avec la liste des 3 documents.


🔍 BONUS: RECHERCHER UN DOCUMENT
─────────────────────────────────

1. Cliquez droit sur la table
2. Menu contextuel → "Papier de travail" → "🔍 Rechercher document"
3. Entrez "I20" ou "inventaire"

Les résultats s'affichent dans la barre latérale.

   → Vérifiez l'orthographe du terme de recherche
