🎯 ARCHITECTURE DU WORKFLOW

Le workflow est composé de 14 nodes :

1. Cross References Form (Form Trigger)
   └─> Formulaire d'upload avec 6 champs

2. Rechercher Dossier CAC (HTTP Request)
   └─> Cherche si "Dossier CAC" existe dans Google Drive

3. Dossier CAC existe? (IF)
   ├─> OUI : Continue vers recherche dossier client
   └─> NON : Crée le dossier CAC

4. Créer Dossier CAC (HTTP Request)
   └─> Crée le dossier "Dossier CAC" à la racine

5. Préparer Recherche Client (Code)
   └─> Prépare les données pour la recherche

6. Rechercher Dossier Client (HTTP Request)
   └─> Cherche le dossier "Client-Exercice"

7. Dossier Client existe? (IF)
   ├─> OUI : Continue vers recherche cycle
   └─> NON : Crée le dossier client + cycles

8. Créer Dossier Client (HTTP Request)
   └─> Crée le dossier "Client-Exercice"

9. Préparer Création Cycles (Code)
   └─> Génère 10 items pour les 10 cycles

10. Créer Cycles (HTTP Request)
    └─> Crée les 10 dossiers de cycles comptables

11. Préparer Recherche Cycle (Code)
    └─> Prépare les données pour la recherche du cycle

12. Rechercher Cycle (HTTP Request)
    └─> Cherche le dossier du cycle comptable

13. Préparer Upload (Code)
    └─> Prépare le nom du fichier et les données binaires

14. Upload Fichier (HTTP Request)
    └─> Upload le fichier dans le dossier du cycle

⚠️ NOTES IMPORTANTES

1. CREDENTIALS OAUTH2
   - Vous DEVEZ configurer les credentials OAuth2 pour chaque node HTTP Request
   - Sans cela, les requêtes échoueront avec une erreur 401 Unauthorized

2. SCOPES GOOGLE DRIVE
   - Assurez-vous que les scopes incluent :
     * https://www.googleapis.com/auth/drive
     * https://www.googleapis.com/auth/drive.file

3. PREMIÈRE EXÉCUTION
   - La première fois, tous les dossiers seront créés
   - Les exécutions suivantes réutiliseront les dossiers existants

4. GESTION DES ERREURS
   - Si un dossier existe déjà, il est réutilisé
   - Si un fichier avec le même nom existe, il sera écrasé

🔍 DÉPANNAGE

PROBLÈME : Erreur 401 Unauthorized
SOLUTION : Vérifier que les credentials OAuth2 sont bien configurés

PROBLÈME : Erreur 403 Forbidden
SOLUTION : Vérifier que les scopes Google Drive sont corrects

PROBLÈME : Le fichier n'est pas uploadé
SOLUTION : Vérifier que le node "Préparer Upload" récupère bien les binary data

PROBLÈME : Les dossiers ne sont pas créés
SOLUTION : Vérifier les permissions du compte Google
