🔗 INTÉGRATION AVEC MENU.JS
───────────────────────────

Le composant communique avec menu.js via des événements personnalisés:

Événements émis par menu.js:
- 'xref:open'  : Ouvrir la sidebar avec des documents
- 'xref:close' : Fermer la sidebar

Événements émis par le composant:
- 'xref:closed'          : Sidebar fermée
- 'xref:document:opened' : Document ouvert

Exemple d'intégration dans menu.js:
```javascript
afficherXRefDocumentaire() {
  const documents = this.extractXRefDocuments();
  
  document.dispatchEvent(new CustomEvent('xref:open', {
    detail: { documents }
  }));
}
```

💡 EXEMPLES DE CODE
───────────────────

Exemple 1: Affichage simple
```typescript
const [isOpen, setIsOpen] = useState(false);
const documents = [
  {
    crossRef: 'I20',
    document: 'Procès verbal inventaire',
    client: 'Orange',
    exercice: '2025',
    cycle: 'Trésorerie'
  }
];

<XRefSidebar
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
  documents={documents}
/>
```

Exemple 2: Avec recherche
```typescript
document.dispatchEvent(new CustomEvent('xref:open', {
  detail: {
    documents: documents,
    searchTerm: 'inventaire',
    title: 'Résultats pour "inventaire"'
  }
}));
```

Exemple 3: Avec callbacks
```typescript
<XRefSidebar
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
  documents={documents}
  onDocumentClick={async (doc) => {
    const response = await fetch('https://...', {
      method: 'POST',
      body: JSON.stringify({ index: doc.crossRef })
    });
    const result = await response.json();
    window.open(result.fileUrl);
  }}
/>
```
