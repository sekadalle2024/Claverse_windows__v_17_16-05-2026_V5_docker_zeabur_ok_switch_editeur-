💡 HYPOTHÈSES PRINCIPALES
═══════════════════════════════════════════════════════════════════════════════

1. STRUCTURE HTML DIFFÉRENTE
   La table dans Claraverse n'a peut-être pas de <thead>
   → Les en-têtes sont dans le <tbody>
   → Utilisation de <td> au lieu de <th>

2. NOM DE COLONNE DIFFÉRENT
   La colonne pourrait s'appeler:
   → "Cross reference" (singulier)
   → "Xref" (sans tiret)
   → "X Ref" (avec espace)

3. MAUVAISE TABLE SÉLECTIONNÉE
   Plusieurs tables dans la page
   → La table X-Ref n'est pas celle qui est active

CODE ACTUEL:

const headerCells = Array.from(rows[0].querySelectorAll('th, td'));
let crossRefIndex = -1;
let documentIndex = -1;

headerCells.forEach((cell, index) => {
  const text = cell.textContent.trim().toLowerCase();
  if ((text.includes('cross') && text.includes('ref')) || 
      text.includes('x-ref') || text.includes('xref')) {
    crossRefIndex = index;
  }
  if (text.includes('document') || text === 'doc') {
    documentIndex = index;
  }
});
