# Guide du Bouton Démarrer - E-audit Pro

## Présentation complète du bouton d'automatisation et de ses modules

Ce guide vous explique comment utiliser le bouton "Démarrer" d'E-audit Pro pour automatiser la génération des étapes de mission d'audit sans avoir à saisir manuellement des commandes complexes.

---

## 🎯 Qu'est-ce que le bouton Démarrer ?

### Définition et fonction
Le bouton "Démarrer" est un outil d'automatisation intelligent qui :
- **Remplace la saisie manuelle** de commandes dans la zone de chat
- **Génère automatiquement** les étapes de mission d'audit
- **Utilise l'intelligence artificielle** pour créer des documents conformes
- **Accélère considérablement** le processus de création

### Localisation sur l'interface
- **Position** : En bas à droite de l'écran
- **Couleur** : Rouge bordeaux pour être facilement repérable
- **Icône** : Symbole de lecture (triangle pointant vers la droite)
- **Texte** : "Démarrer" clairement visible

---

## 🖥️ Interface et navigation

### Accès au bouton Démarrer
1. **Localiser** le bouton rouge bordeaux en bas à droite
2. **Cliquer** une fois pour ouvrir le menu déroulant
3. **Observer** les 4 modules qui s'affichent
4. **Sélectionner** le module approprié à votre besoin

### Menu déroulant - Les 4 modules

#### 1. E-audit pro
- **Icône** : Logo E-audit avec flèche vers la droite
- **Fonction** : Module principal pour les missions d'audit interne
- **Utilisation** : Génération de programmes de travail, FRAP, rapports d'audit
- **Public cible** : Auditeurs internes, chefs de mission

#### 2. E-cartographie  
- **Icône** : Symbole de cartographie/réseau
- **Fonction** : Outil de cartographie des risques et processus
- **Utilisation** : Création de matrices de risques, cartographies de processus
- **Public cible** : Risk managers, directeurs d'audit

#### 3. E-revision
- **Icône** : Symbole comptable/révision
- **Fonction** : Module spécialisé pour la révision comptable
- **Utilisation** : Contrôles des comptes, tests de détail, revue analytique
- **Public cible** : Experts-comptables, commissaires aux comptes

#### 4. Bibliothèque
- **Icône** : Symbole de livre/ressources
- **Fonction** : Centre de ressources avec 1000 contrôles et modèles
- **Utilisation** : Consultation de référentiels, recherche de contrôles types
- **Public cible** : Tous les utilisateurs pour documentation

---

## 🔧 Processus d'automatisation

### Étape 1 : Sélection du module
1. **Cliquer** sur le bouton "Démarrer"
2. **Analyser** votre besoin :
   - Mission d'audit interne → E-audit pro
   - Cartographie des risques → E-cartographie  
   - Révision comptable → E-revision
   - Recherche de ressources → Bibliothèque
3. **Cliquer** sur le module choisi

### Étape 2 : Sous-menu spécialisé (exemple E-revision)
Après avoir cliqué sur "E-revision", un sous-menu s'ouvre avec :
- **Contrôle des comptes** : Pour les tests sur les comptes annuels
- **Revue analytique** : Pour les analyses de cohérence
- **Tests de détail** : Pour les contrôles substantifs approfondis
- **Bibliothèque** : Accès aux ressources spécialisées

### Étape 3 : Sélection de l'étape spécifique
Dans le sous-menu "Tests de détail", vous trouvez :
- **Normal** : Tests standards selon les normes
- **Avancé** : Tests approfondis pour risques élevés
- **Intelligent** : Tests adaptatifs selon le contexte

### Étape 4 : Génération automatique
1. **Sélectionner** le niveau souhaité (Normal/Avancé/Intelligent)
2. **L'IA analyse** votre contexte de mission
3. **Génération automatique** des commandes appropriées
4. **Création** du document conforme aux normes

## 📋 Utilisation pratique par module

### E-audit pro - Cas d'usage
**Situation** : Vous devez créer un programme de travail pour auditer la paie
1. **Clic** sur "Démarrer" → "E-audit pro"
2. **Sélection** "Programme de travail"
3. **Paramétrage** : Processus "Paie", Durée "1 semaine"
4. **Génération** : Programme avec 20 contrôles adaptés créé automatiquement

### E-cartographie - Cas d'usage  
**Situation** : Vous devez cartographier les risques du processus achats
1. **Clic** sur "Démarrer" → "E-cartographie"
2. **Sélection** "Cartographie des risques"
3. **Paramétrage** : Processus "Achats", Niveau "Détaillé"
4. **Génération** : Matrice complète avec 15 risques identifiés

### E-revision - Cas d'usage
**Situation** : Vous devez effectuer des tests de détail sur les créances
1. **Clic** sur "Démarrer" → "E-revision" → "Tests de détail"
2. **Sélection** "Avancé" (pour risque élevé)
3. **Paramétrage** : Compte "Créances clients", Seuil "50K€"
4. **Génération** : Programme de tests avec 12 contrôles spécialisés

### Bibliothèque - Cas d'usage
**Situation** : Vous cherchez des contrôles types sur la trésorerie
1. **Clic** sur "Démarrer" → "Bibliothèque"
2. **Recherche** : "Trésorerie" dans la barre de recherche
3. **Filtrage** : Par niveau de risque, secteur d'activité
4. **Sélection** : 25 contrôles pertinents identifiés

## 🛠️ Résolution de problèmes

### Problèmes fréquents
- **Menu ne s'ouvre pas** : Vérifier la connexion internet
- **Génération échoue** : Paramètres incomplets ou incorrects
- **Résultat inadapté** : Mauvais choix de module ou niveau
- **Lenteur** : Surcharge temporaire du serveur

### Solutions recommandées
1. **Actualiser la page** si le bouton ne répond pas
2. **Vérifier les paramètres** avant de lancer la génération
3. **Choisir le bon niveau** (Normal/Avancé/Intelligent)
4. **Contacter le support** si problème persistant
