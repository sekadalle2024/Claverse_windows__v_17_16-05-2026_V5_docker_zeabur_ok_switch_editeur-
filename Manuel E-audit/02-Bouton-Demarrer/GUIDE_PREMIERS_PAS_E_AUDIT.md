# Guide des Premiers Pas - E-audit Pro

## Vos 30 premières minutes avec E-audit Pro

Ce guide vous accompagne dans vos tout premiers pas après votre connexion réussie à E-audit Pro. Suivez ces étapes pour une prise en main optimale.

---

## Étape 1 : Découverte du tableau de bord (5 minutes)

### Ce que vous voyez en arrivant
Après votre connexion, vous arrivez sur le **tableau de bord principal**. C'est votre espace de travail central.

### Éléments principaux à identifier :
- **Barre de navigation** : En haut, vos menus principaux
- **Zone centrale** : Vos missions en cours et statistiques
- **Panneau latéral** : Accès rapide aux fonctions courantes
- **Votre profil** : En haut à droite, vos informations personnelles

### Première action recommandée :
1. Cliquez sur votre nom en haut à droite
2. Vérifiez que vos informations sont correctes
3. Notez votre niveau d'accès (Auditeur, Chef de mission, etc.)

---

## Étape 2 : Explorer les menus principaux (10 minutes)

### Menu "Missions"
- **Mes missions** : Toutes vos missions d'audit (vide pour l'instant)
- **Nouvelle mission** : Créer une nouvelle mission d'audit
- **Missions partagées** : Missions où vous collaborez avec d'autres

### Menu "Référentiels"
- **Contrôles standards** : La base de 1000 contrôles d'E-audit Pro
- **Mes référentiels** : Vos contrôles personnalisés
- **Bibliothèque** : Modèles et exemples

### Menu "Rapports"
- **Mes rapports** : Tous vos rapports générés
- **Modèles** : Templates de rapports disponibles
- **Archives** : Rapports des missions terminées

### Menu "Suivi"
- **Recommandations** : Suivi des plans d'action
- **Tableaux de bord** : Statistiques et indicateurs
- **Calendrier** : Planning de vos missions

---

## Étape 3 : Créer votre première mission de test (15 minutes)

### Pourquoi créer une mission de test ?
- Découvrir l'interface sans pression
- Comprendre le processus complet
- Tester les fonctionnalités principales
- Vous familiariser avec la terminologie

### Étapes de création :

#### 3.1 Lancer la création
1. Cliquez sur **"Nouvelle mission"**
2. Choisissez **"Mission de formation"** ou **"Mission test"**

#### 3.2 Remplir les informations de base
- **Nom de la mission** : "Ma première mission - Formation"
- **Type d'audit** : Choisissez "Audit de processus"
- **Cycle d'activité** : Sélectionnez "Achats/Fournisseurs" (le plus simple)
- **Période auditée** : Mois en cours
- **Durée prévue** : 2 jours

#### 3.3 Sélectionner les contrôles
1. E-audit Pro vous propose automatiquement des contrôles pour le cycle "Achats"
2. Gardez les contrôles pré-sélectionnés (environ 10-15 contrôles)
3. Cliquez sur **"Valider la sélection"**

#### 3.4 Finaliser la création
1. Vérifiez le résumé de votre mission
2. Cliquez sur **"Créer la mission"**
3. Votre mission apparaît maintenant dans "Mes missions"

---

## Étape 4 : Explorer votre mission de test

### Accéder à votre mission
1. Allez dans **"Mes missions"**
2. Cliquez sur votre mission "Ma première mission - Formation"
3. Vous arrivez sur la page de gestion de mission

### Découvrir les onglets de mission

#### Onglet "Vue d'ensemble"
- Informations générales de la mission
- Statut d'avancement
- Équipe assignée

#### Onglet "Programme de travail"
- Liste de tous les contrôles à effectuer
- Statut de chaque contrôle (À faire, En cours, Terminé)
- Temps estimé par contrôle

#### Onglet "Feuilles de test"
- Interface pour saisir vos constats
- Zone de commentaires
- Upload de pièces justificatives

#### Onglet "FRAP"
- Feuilles de Révélation et d'Analyse des Problèmes
- Génération automatique basée sur vos constats
- Possibilité de modification manuelle

#### Onglet "Rapport"
- Rapport automatiquement généré
- Export en PDF ou Word
- Historique des versions

---

## Étape 5 : Tester un contrôle simple

### Choisir un contrôle facile
1. Allez dans l'onglet **"Programme de travail"**
2. Sélectionnez un contrôle simple comme "Vérification de l'existence des bons de commande"
3. Cliquez sur **"Commencer ce contrôle"**

### Simuler un test
1. Dans la feuille de test, vous voyez :
   - **Objectif du contrôle** : Ce que vous devez vérifier
   - **Procédure** : Comment faire le test
   - **Zone de saisie** : Pour vos constats

2. Pour cet exercice, saisissez un constat fictif :
   - **Échantillon testé** : "10 bons de commande"
   - **Résultat** : "Conforme" ou "Non conforme"
   - **Commentaire** : "Test de formation - Tous les bons sont signés"

3. Cliquez sur **"Sauvegarder"**

### Observer la génération automatique
1. Allez dans l'onglet **"FRAP"**
2. Vous voyez qu'E-audit Pro a automatiquement créé une fiche basée sur votre constat
3. Si vous avez mis "Non conforme", une recommandation a été générée

---

## Étape 6 : Découvrir les fonctionnalités avancées

### Référentiels de contrôles
1. Allez dans **"Référentiels" > "Contrôles standards"**
2. Explorez les différents cycles d'activité
3. Regardez la structure : Domaine > Objectif > Risque > Contrôle

### Modèles de rapports
1. Allez dans **"Rapports" > "Modèles"**
2. Découvrez les différents formats disponibles
3. Notez les éléments personnalisables

### Tableaux de bord
1. Allez dans **"Suivi" > "Tableaux de bord"**
2. Même avec une seule mission test, vous voyez des statistiques
3. Comprenez les indicateurs proposés

---

## Étape 7 : Personnaliser votre environnement

### Paramètres utilisateur
1. Cliquez sur votre nom en haut à droite
2. Allez dans **"Paramètres"**
3. Personnalisez :
   - Vos préférences d'affichage
   - Vos notifications
   - Votre signature pour les rapports

### Favoris et raccourcis
1. Ajoutez vos fonctions les plus utilisées en favoris
2. Personnalisez votre tableau de bord
3. Configurez vos alertes

---

## Étape 8 : Préparer votre première vraie mission

### Réflexion préparatoire
Avant de créer votre première vraie mission, réfléchissez à :
- **Quel processus auditer** : Choisissez quelque chose que vous connaissez bien
- **Quelle période** : Commencez par une période récente et courte
- **Quels objectifs** : Définissez 2-3 objectifs clairs
- **Quelles ressources** : Identifiez vos interlocuteurs et documents

### Checklist avant de commencer
- [ ] J'ai exploré l'interface générale
- [ ] J'ai testé la création d'une mission
- [ ] J'ai compris le principe des feuilles de test
- [ ] J'ai vu comment les rapports se génèrent
- [ ] J'ai identifié les référentiels utiles pour mon secteur
- [ ] J'ai configuré mes paramètres personnels

## Conseils pour bien continuer

### Bonnes pratiques découvertes
1. **Sauvegardez régulièrement** : E-audit Pro sauvegarde automatiquement, mais prenez l'habitude
2. **Utilisez les commentaires** : Plus vous documentez, meilleur sera le rapport automatique
3. **Explorez les référentiels** : Ils contiennent une richesse de bonnes pratiques
4. **Personnalisez progressivement** : Adaptez l'outil à votre façon de travailler

### Prochaines étapes recommandées
1. **Semaine 1** : Créez votre première vraie mission sur un processus simple
2. **Semaine 2** : Testez l'import de vos anciens programmes de travail
3. **Semaine 3** : Explorez les fonctions de collaboration si vous travaillez en équipe
4. **Mois 1** : Maîtrisez la personnalisation des rapports

### Ressources pour approfondir
- **Guide d'onboarding complet** : Pour une formation structurée
- **FAQ** : Pour résoudre les questions courantes
- **Support technique** : Pour l'aide personnalisée
- **Communauté utilisateurs** : Pour échanger avec d'autres auditeurs
