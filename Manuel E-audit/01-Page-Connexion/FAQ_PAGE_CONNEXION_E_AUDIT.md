# FAQ - Page de Connexion E-audit Pro

## 🔐 Questions sur la Connexion

### 1. **Comment obtenir mes identifiants de connexion ?**
**Réponse :** Vos identifiants vous sont fournis par votre responsable informatique ou l'administrateur E-audit Pro de votre organisation. Si vous n'avez pas reçu vos identifiants, contactez votre service IT ou votre responsable audit interne.

### 2. **Que faire si j'ai oublié mon mot de passe ?**
**Réponse :** Actuellement, contactez votre administrateur système pour réinitialiser votre mot de passe. Une fonction "Mot de passe oublié" sera bientôt disponible directement sur cette page.

### 3. **Pourquoi ma connexion échoue-t-elle ?**
**Réponse :** Vérifiez ces points dans l'ordre :
- L'orthographe exacte de votre email (pas d'espace avant/après)
- Votre mot de passe (attention aux majuscules/minuscules)
- Votre connexion internet
- Si le problème persiste, contactez le support technique

### 4. **Puis-je utiliser mon email personnel pour me connecter ?**
**Réponse :** Non, utilisez uniquement votre adresse email professionnelle. C'est avec cette adresse que votre compte a été créé et que vous recevrez les notifications importantes.

### 5. **Combien de tentatives de connexion ai-je droit ?**
**Réponse :** Après 5 tentatives incorrectes, votre compte sera temporairement bloqué pour des raisons de sécurité. Attendez 15 minutes ou contactez votre administrateur pour débloquer votre compte.

## 👤 Questions sur la Création de Compte

### 6. **Puis-je créer un compte sans autorisation de mon entreprise ?**
**Réponse :** Non, E-audit Pro est un logiciel professionnel. La création de compte doit être validée par votre organisation. Demandez d'abord l'autorisation à votre responsable avant de créer un compte.

### 7. **Quelles informations dois-je fournir pour créer un compte ?**
**Réponse :** Vous devrez renseigner :
- Vos nom et prénom complets
- Votre email professionnel
- Votre fonction dans l'organisation
- Le nom de votre entreprise/service
- Un mot de passe sécurisé

### 8. **Comment créer un mot de passe sécurisé ?**
**Réponse :** Votre mot de passe doit contenir :
- Au moins 8 caractères
- Des lettres majuscules et minuscules
- Des chiffres
- Des caractères spéciaux (@, #, !, etc.)
Exemple : "Audit2024#Secure"

### 9. **Vais-je recevoir un email de confirmation après inscription ?**
**Réponse :** Oui, vous recevrez un email de confirmation sur votre adresse professionnelle. Vérifiez aussi vos spams. Cliquez sur le lien de confirmation pour activer votre compte.

### 10. **Combien de temps faut-il pour que mon compte soit actif ?**
**Réponse :** Après validation de votre email, votre compte est généralement actif immédiatement. Dans certaines organisations, un administrateur doit d'abord approuver votre demande (délai de 24-48h).

## 🔧 Questions Techniques

### 11. **Sur quels navigateurs E-audit Pro fonctionne-t-il ?**
**Réponse :** E-audit Pro fonctionne sur :
- Google Chrome (recommandé)
- Mozilla Firefox
- Microsoft Edge
- Safari (Mac)
Utilisez toujours la version la plus récente de votre navigateur.

### 12. **Puis-je utiliser E-audit Pro sur mon téléphone ?**
**Réponse :** E-audit Pro est optimisé pour les ordinateurs et tablettes. L'utilisation sur smartphone est possible mais limitée. Pour une expérience optimale, utilisez un écran d'au moins 10 pouces.

### 13. **Ai-je besoin d'installer un logiciel sur mon ordinateur ?**
**Réponse :** Non, E-audit Pro fonctionne entièrement dans votre navigateur web. Aucune installation n'est nécessaire. Il vous suffit d'avoir une connexion internet.

### 14. **Que faire si la page ne s'affiche pas correctement ?**
**Réponse :** Essayez ces solutions :
- Actualisez la page (F5 ou Ctrl+F5)
- Videz le cache de votre navigateur
- Désactivez temporairement les extensions de navigateur
- Essayez un autre navigateur

### 15. **Ma connexion internet est lente, puis-je quand même utiliser E-audit Pro ?**
**Réponse :** E-audit Pro fonctionne avec une connexion internet basique. Cependant, pour une expérience fluide, nous recommandons une connexion d'au moins 2 Mbps. Les gros fichiers (rapports PDF) peuvent prendre plus de temps à télécharger.

## 🛡️ Questions de Sécurité

### 16. **Mes données sont-elles sécurisées sur E-audit Pro ?**
**Réponse :** Oui, E-audit Pro utilise :
- Cryptage SSL/TLS pour toutes les communications
- Stockage sécurisé de vos données
- Sauvegardes automatiques régulières
- Accès restreint selon vos droits utilisateur

### 17. **Combien de temps ma session reste-t-elle active ?**
**Réponse :** Pour votre sécurité, votre session expire automatiquement après 30 minutes d'inactivité. Vous devrez vous reconnecter. Sauvegardez régulièrement votre travail.

### 18. **Puis-je partager mes identifiants avec un collègue ?**
**Réponse :** Non, jamais ! Chaque utilisateur doit avoir ses propres identifiants. Le partage de comptes :
- Compromet la sécurité
- Empêche la traçabilité des actions
- Viole les conditions d'utilisation

### 19. **Que se passe-t-il si je me connecte depuis plusieurs endroits ?**
**Réponse :** Vous pouvez vous connecter depuis différents ordinateurs, mais une seule session active est autorisée à la fois. Si vous vous connectez ailleurs, votre première session sera automatiquement fermée.

### 20. **Comment me déconnecter correctement ?**
**Réponse :** Utilisez toujours le bouton "Déconnexion" dans l'application avant de fermer votre navigateur. Cela garantit :
- La sauvegarde de vos travaux en cours
- La fermeture sécurisée de votre session
- La protection de vos données
