# Guide de Présentation - Page de Connexion E-audit Pro

## Vue d'ensemble de la page

Cette page est la porte d'entrée de votre logiciel E-audit Pro. C'est la première interface que vous verrez à chaque fois que vous souhaitez utiliser l'application.

---

## Éléments présents sur la page

### 1. Logo de l'application
En haut de la page, vous voyez le logo d'E-audit avec son icône distinctive orange et jaune. Ce logo vous confirme que vous êtes bien sur l'application E-audit Pro.

### 2. Titre de bienvenue
**"Bienvenue sur E-audit"**

Ce message vous accueille et vous indique que vous êtes au bon endroit pour commencer votre travail d'audit.

### 3. Sous-titre explicatif
**"Connectez-vous ou créez un nouveau compte"**

Ce texte vous explique les deux possibilités qui s'offrent à vous :
- Si vous avez déjà un compte, vous pouvez vous connecter
- Si c'est votre première visite, vous pouvez créer un nouveau compte

---

## Champs à remplir

### Champ 1 : Email
- **Libellé** : "Email"
- **Icône** : Une enveloppe à gauche du champ
- **Texte d'exemple** : "votre.email@exemple.com"
- **Action** : Cliquez dans ce champ et tapez votre adresse email professionnelle

**Pourquoi c'est important** : Votre email sert d'identifiant unique. C'est avec cette adresse que le système vous reconnaîtra à chaque connexion.

### Champ 2 : Mot de passe
- **Libellé** : "Mot de passe"
- **Icône** : Un cadenas à gauche du champ
- **Texte d'exemple** : "Votre mot de passe"
- **Icône à droite** : Un œil qui permet de voir ou masquer votre mot de passe
- **Action** : Cliquez dans ce champ et tapez votre mot de passe

**Astuce pratique** : Si vous cliquez sur l'icône de l'œil à droite, vous pourrez voir les caractères que vous tapez. Cela vous aide à vérifier que vous n'avez pas fait d'erreur de frappe.

---

## Boutons d'action disponibles

### Bouton principal : "Se connecter"
- **Couleur** : Rose vif (impossible à manquer)
- **Icône** : Une flèche pointant vers la droite
- **Action** : Cliquez sur ce bouton après avoir rempli votre email et mot de passe

**Que se passe-t-il après** : 
- Si vos identifiants sont corrects, vous serez dirigé vers le tableau de bord principal d'E-audit Pro
- Si vos identifiants sont incorrects, un message d'erreur apparaîtra vous demandant de vérifier vos informations

### Bouton secondaire : "Créer un nouveau compte"
- **Couleur** : Blanc avec bordure grise
- **Icône** : Un symbole de personne avec un "+"
- **Action** : Cliquez sur ce bouton si vous n'avez pas encore de compte

**Que se passe-t-il après** :
- Vous serez dirigé vers une page d'inscription
- On vous demandera de renseigner vos informations : nom, prénom, email, mot de passe, organisation
- Une fois l'inscription terminée, vous pourrez vous connecter avec vos nouveaux identifiants

### Bouton tertiaire : "Accès Administrateur"
- **Couleur** : Gris clair
- **Icône** : Un bouclier de sécurité
- **Action** : Ce bouton est réservé aux administrateurs du système

**Important** : Ce bouton n'est pas pour les utilisateurs standards. Il permet aux administrateurs de :
- Gérer les comptes utilisateurs
- Configurer les paramètres globaux de l'application
- Accéder aux fonctionnalités de supervision

---

## Parcours utilisateur typique

### Scénario 1 : Première utilisation (Nouvel utilisateur)
1. Vous arrivez sur cette page
2. Vous cliquez sur "Créer un nouveau compte"
3. Vous remplissez le formulaire d'inscription
4. Vous recevez une confirmation (par email ou à l'écran)
5. Vous revenez sur cette page de connexion
6. Vous entrez votre email et mot de passe
7. Vous cliquez sur "Se connecter"
8. **→ Vous accédez au tableau de bord principal**

### Scénario 2 : Utilisation quotidienne (Utilisateur existant)
1. Vous arrivez sur cette page
2. Vous entrez votre email
3. Vous entrez votre mot de passe
4. Vous cliquez sur "Se connecter"
5. **→ Vous accédez directement au tableau de bord principal**

Après une connexion réussie, vous arriverez sur le tableau de bord principal d'E-audit Pro où vous pourrez :

## Questions fréquentes

### J'ai oublié mon mot de passe, que faire ?
Actuellement, cette page ne montre pas de lien "Mot de passe oublié". Contactez votre administrateur système ou le support technique d'E-audit Pro pour réinitialiser votre mot de passe.

### Puis-je utiliser E-audit Pro sur plusieurs appareils ?
Oui, vous pouvez vous connecter depuis n'importe quel ordinateur ou tablette avec un navigateur web. Vos données sont sauvegardées en ligne et accessibles partout.

### Combien de temps ma session reste-t-elle active ?
Pour des raisons de sécurité, si vous ne faites aucune action pendant un certain temps (généralement 30 minutes à 1 heure), vous serez automatiquement déconnecté. Il vous suffira de vous reconnecter.

### Mes données sont-elles sécurisées ?
Oui, E-audit Pro utilise des protocoles de sécurité standards pour protéger vos informations. Votre mot de passe est crypté et vos données d'audit sont stockées de manière sécurisée.

---

## Conseils pratiques pour bien démarrer

1. **Utilisez un mot de passe fort** : Mélangez lettres majuscules, minuscules, chiffres et caractères spéciaux
2. **Notez vos identifiants** : Conservez-les dans un endroit sûr (gestionnaire de mots de passe recommandé)
3. **Vérifiez votre connexion internet** : E-audit Pro est une application en ligne qui nécessite une connexion stable
4. **Utilisez un navigateur récent** : Chrome, Firefox, Edge ou Safari dans leurs versions récentes
