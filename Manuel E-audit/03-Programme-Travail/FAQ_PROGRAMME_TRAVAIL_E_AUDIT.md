## 📋 Questions sur la Création du Programme

### 1. **Comment accéder à la création du programme de travail ?**
**Réponse :** Deux méthodes s'offrent à vous :
- **Méthode classique** : Menu "Missions" > Sélectionner votre mission > Onglet "Programme"
- **Méthode rapide** : Bouton "Démarrer" (rouge bordeaux) > "E-audit pro" > "Programme de travail"
La méthode avec le bouton Démarrer est recommandée car elle génère automatiquement un programme adapté à votre processus.

### 2. **Quelle est la différence entre génération automatique et création manuelle ?**
**Réponse :** 
- **Génération automatique** : E-audit Pro sélectionne 15-25 contrôles adaptés à votre processus en 2 minutes. Idéal pour débuter ou gagner du temps.
- **Création manuelle** : Vous choisissez chaque contrôle dans la bibliothèque de 1000 contrôles. Plus long mais totalement personnalisé.
Conseil : Commencez par la génération automatique, puis personnalisez selon vos besoins.

### 3. **Combien de contrôles dois-je inclure dans mon programme ?**
**Réponse :** Cela dépend de plusieurs facteurs :
- **Mission simple** (2-3 jours) : 10-15 contrôles
- **Mission standard** (1 semaine) : 15-25 contrôles  
- **Mission complexe** (2+ semaines) : 25-40 contrôles
L'important n'est pas la quantité mais la pertinence. Mieux vaut 15 contrôles bien choisis que 30 contrôles superficiels.

### 4. **Comment savoir si mon programme couvre tous les risques ?**
**Réponse :** E-audit Pro vous aide avec :
- **Indicateur de couverture** : Pourcentage des risques identifiés couverts par vos contrôles
- **Alertes automatiques** : Signalement des risques majeurs non couverts
- **Matrice de couverture** : Vue croisée risques/contrôles
Visez 100% de couverture des risques élevés et 80% des risques moyens.

### 5. **Puis-je réutiliser un programme d'une mission précédente ?**
**Réponse :** Oui, plusieurs options :
- **Dupliquer une mission** : Copie complète du programme
- **Utiliser comme modèle** : Sauvegarde d'un programme réussi
- **Import partiel** : Sélectionner certains contrôles d'anciennes missions
Attention : Adaptez toujours le programme au contexte spécifique de la nouvelle mission.

## 🔧 Questions sur la Personnalisation

### 6. **Comment adapter les contrôles standards à mon organisation ?**
**Réponse :** Chaque contrôle peut être personnalisé :
- **Libellés** : Modifier le texte pour utiliser votre terminologie
- **Documents** : Préciser les documents spécifiques de votre organisation
- **Seuils** : Ajuster les montants de matérialité
- **Échantillons** : Adapter la taille selon vos volumes
Cliquez sur l'icône "Modifier" à droite de chaque contrôle.

### 7. **Puis-je créer mes propres contrôles personnalisés ?**
**Réponse :** Absolument ! Utilisez le bouton "Nouveau contrôle" :
1. Définissez la tâche clé à auditer
2. Identifiez le point de contrôle spécifique
3. Précisez le risque couvert
4. Décrivez l'action d'audit à réaliser
5. Indiquez les documents à examiner
Vos contrôles personnalisés sont sauvegardés pour réutilisation future.

### 8. **Comment organiser l'ordre des contrôles dans mon programme ?**
**Réponse :** Plusieurs approches possibles :
- **Par processus** : Regrouper par étapes du processus audité
- **Par risque** : Commencer par les risques les plus élevés
- **Par logique d'audit** : Tests de conformité puis tests substantifs
- **Par praticité** : Optimiser les déplacements et entretiens
Utilisez le glisser-déposer pour réorganiser facilement.

### 9. **Comment estimer le temps nécessaire pour chaque contrôle ?**
**Réponse :** E-audit Pro propose des estimations basées sur :
- **Complexité du contrôle** : Simple (30 min), Moyen (1h), Complexe (2h+)
- **Taille de l'échantillon** : Plus l'échantillon est grand, plus c'est long
- **Disponibilité des documents** : Temps d'attente si documents à rassembler
- **Expérience de l'auditeur** : Junior vs Senior
Ajustez selon votre contexte spécifique.

### 10. **Puis-je assigner des contrôles à différents membres de l'équipe ?**
**Réponse :** Oui, dans la colonne "Assigné à" :
- Sélectionnez le membre de l'équipe pour chaque contrôle
- Équilibrez la charge de travail entre les auditeurs
- Tenez compte des compétences spécialisées
- Prévoyez des contrôles croisés pour validation
L'équipe recevra automatiquement ses affectations par email.

## 📊 Questions sur la Validation et le Suivi

### 11. **Comment valider mon programme avant de commencer les tests ?**
**Réponse :** Processus de validation en 3 étapes :
1. **Auto-contrôle** : E-audit Pro vérifie la cohérence automatiquement
2. **Validation hiérarchique** : Soumission à votre responsable via le bouton "Soumettre"
3. **Validation finale** : Approbation et verrouillage du programme
Le statut passe de "Brouillon" à "Validé" puis "Actif".

### 12. **Que faire si mon responsable demande des modifications ?**
**Réponse :** Le processus de révision est simple :
- Votre responsable ajoute des commentaires sur les contrôles à modifier
- Vous recevez une notification avec les demandes de changement
- Effectuez les modifications demandées
- Resoumettez pour validation
L'historique des versions est conservé pour traçabilité.

### 13. **Puis-je modifier le programme une fois la mission commencée ?**
**Réponse :** Oui, mais avec des restrictions :
- **Ajouts** : Nouveaux contrôles possibles avec justification
- **Suppressions** : Uniquement si contrôle non encore réalisé
- **Modifications** : Adaptation des contrôles en cours
- **Traçabilité** : Toutes les modifications sont documentées
Demandez validation hiérarchique pour les changements majeurs.

### 14. **Comment suivre l'avancement de l'exécution du programme ?**
**Réponse :** Le tableau de bord vous donne une vue temps réel :
- **Statut par contrôle** : Non commencé, En cours, Terminé
- **Pourcentage global** : Avancement de la mission
- **Alertes** : Retards ou problèmes identifiés
- **Temps consommé** : Comparaison avec les estimations
Accessible depuis l'onglet "Suivi" de votre mission.

### 15. **Comment exporter mon programme pour le partager ?**
**Réponse :** Plusieurs formats d'export disponibles :
- **PDF** : Document final non modifiable pour diffusion
- **Word** : Version modifiable pour présentation
- **Excel** : Données brutes pour analyse
- **Email** : Envoi direct aux parties prenantes
Utilisez le bouton "Exporter" en haut à droite de l'interface.

## 🛠️ Questions Techniques

### 16. **Puis-je importer un programme Excel existant ?**
**Réponse :** Oui, via la fonction d'import :
1. Téléchargez le modèle Excel d'E-audit Pro
2. Remplissez vos contrôles selon le format demandé
3. Importez le fichier via "Import" > "Programme Excel"
4. Vérifiez et validez les données importées
Cette fonction est idéale pour migrer vos anciens programmes.

### 17. **Comment sauvegarder mon travail en cours ?**
**Réponse :** E-audit Pro sauvegarde automatiquement :
- **Sauvegarde auto** : Toutes les 30 secondes
- **Sauvegarde manuelle** : Ctrl+S ou bouton "Sauvegarder"
- **Versions** : Historique des modifications conservé
- **Récupération** : Restauration possible en cas de problème
L'indicateur "Sauvegardé" confirme que vos données sont sécurisées.

### 18. **Que faire si l'interface se fige pendant la création ?**
**Réponse :** Solutions par ordre de priorité :
1. **Attendre** 30 secondes (sauvegarde automatique en cours)
2. **Actualiser** la page (F5) - vos données sont conservées
3. **Vider le cache** du navigateur
4. **Changer de navigateur** (Chrome recommandé)
5. **Contacter le support** si le problème persiste

### 19. **Comment collaborer en temps réel avec mon équipe ?**
**Réponse :** Fonctionnalités collaboratives intégrées :
- **Partage** : Inviter des collaborateurs via email
- **Édition simultanée** : Plusieurs personnes peuvent travailler ensemble
- **Commentaires** : Annotations sur chaque contrôle
- **Notifications** : Alertes sur les modifications importantes
- **Chat intégré** : Communication directe dans l'interface

### 20. **Puis-je utiliser E-audit Pro sur tablette pour créer mon programme ?**
**Réponse :** E-audit Pro est optimisé pour les ordinateurs, mais utilisable sur tablette :
- **Écran minimum** : 10 pouces recommandé
- **Fonctionnalités** : Toutes disponibles mais interface adaptée
- **Performance** : Plus lente qu'sur ordinateur
- **Recommandation** : Création sur PC, consultation sur tablette
Pour une expérience optimale, utilisez un ordinateur avec écran large.

## 💡 Conseils Pratiques

### **Raccourcis clavier utiles :**
- **Ctrl + N** : Nouveau contrôle
- **Ctrl + D** : Dupliquer le contrôle sélectionné
- **Ctrl + S** : Sauvegarder
- **Ctrl + Z** : Annuler la dernière action
- **Ctrl + F** : Rechercher dans les contrôles
- **Tab** : Naviguer entre les champs

### **Optimisation de votre workflow :**
- Commencez toujours par la génération automatique
- Personnalisez progressivement selon vos besoins
- Sauvegardez vos programmes réussis comme modèles
- Impliquez votre équipe dans la validation
- Documentez vos choix méthodologiques
- Prévoyez 20% de temps supplémentaire pour les imprévus

### **Indicateurs de qualité d'un bon programme :**
- 100% des risques élevés couverts
- Équilibre entre tests de conformité et substantifs
- Temps total réaliste par rapport aux ressources
- Contrôles adaptés au contexte de l'organisation
- Validation hiérarchique obtenue
- Équipe formée et prête à exécuter
