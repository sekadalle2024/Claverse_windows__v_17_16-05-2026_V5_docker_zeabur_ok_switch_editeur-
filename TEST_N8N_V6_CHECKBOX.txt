📍 OBJECTIF
───────────────────────────────────────────────────────────────
Vérifier que le code n8n V6 détecte correctement l'état des
checkboxes et retourne true/false dans le JSON.


📍 FICHIER À UTILISER
───────────────────────────────────────────────────────────────
n8n-code-javascript3-V6-CHECKBOX.js


📍 TEST 1: CHECKBOX COCHÉE
───────────────────────────────────────────────────────────────

HTML INPUT:

  <input type="checkbox" checked="checked">
</td>

JSON OUTPUT ATTENDU:

  "Reponse User": true
}

───────────────────────────────────────────────────────────────

Dans n8n, vérifier:

📍 EXEMPLE RÉEL
───────────────────────────────────────────────────────────────

Table CIA avec 4 options, Option 3 cochée:

INPUT:
- Option 1: data-checked="false"
- Option 2: data-checked="false"
- Option 3: data-checked="true"  ← Cochée
- Option 4: data-checked="false"

OUTPUT:
[
  { "Option": "Option 1", "Reponse User": false },
  { "Option": "Option 2", "Reponse User": false },
  { "Option": "Option 3", "Reponse User": true },  ← true
  { "Option": "Option 4", "Reponse User": false }
]


📍 TRAITEMENT ULTÉRIEUR
───────────────────────────────────────────────────────────────

Dans n8n, vous pouvez maintenant:

// Filtrer les réponses cochées
$json["table 1"].filter(row => row["Reponse User"] === true)

// Compter les réponses cochées
$json["table 1"].filter(row => row["Reponse User"]).length

// Trouver la réponse cochée
$json["table 1"].find(row => row["Reponse User"])?.Option
