❌ PROBLÈME IDENTIFIÉ
═══════════════════════

Erreur lors du build Docker:
> clara-verse@0.1.25 postinstall
> electron-builder install-app-deps

sh: electron-builder: not found
npm error code 127

🔍 CAUSE RACINE
═══════════════

Le package.json contient un script postinstall (ligne 34):
"postinstall": "electron-builder install-app-deps"

Ce script:
- ✅ Est nécessaire pour les builds Electron (desktop)
- ❌ N'est PAS nécessaire pour le déploiement web
- ❌ Cause l'échec du build Docker car electron-builder n'est pas installé

💡 POURQUOI ELECTRON-BUILDER N'EST PAS INSTALLÉ ?
═════════════════════════════════════════════════

electron-builder est dans devDependencies, mais:
1. npm install installe TOUTES les dépendances (dev + prod)
2. electron-builder s'installe correctement
3. MAIS le postinstall s'exécute PENDANT l'installation
4. À ce moment, electron-builder n'est pas encore dans le PATH
5. Résultat: "electron-builder: not found"

✅ SOLUTION 1 : IGNORER LE POSTINSTALL (RECOMMANDÉ)
═══════════════════════════════════════════════════

Modifier la ligne dans Dockerfile.frontend:

AVANT:
RUN npm install

APRÈS:
RUN npm install --ignore-scripts

Avantages:
✓ Simple et direct
✓ Ignore tous les scripts postinstall
✓ Plus rapide (pas d'exécution de scripts inutiles)
✓ Pas de modification du code source

✅ SOLUTION 2 : VARIABLE D'ENVIRONNEMENT
════════════════════════════════════════

Ajouter avant npm install:

ENV ELECTRON_SKIP_BINARY_DOWNLOAD=1
RUN npm install

Avantages:
✓ Spécifique à Electron
✓ Permet d'autres postinstall scripts de s'exécuter

⚠️ SOLUTION 3 : MODIFIER package.json (NON RECOMMANDÉ)
═══════════════════════════════════════════════════════

Changer le postinstall pour vérifier l'environnement:
"postinstall": "if [ \"$SKIP_ELECTRON\" != \"true\" ]; then electron-builder install-app-deps; fi"

Inconvénients:
✗ Modifie le code source
✗ Peut affecter les développeurs locaux
✗ Plus complexe

🎯 DOCKERFILE CORRIGÉ (SOLUTION 1)
═══════════════════════════════════

# Installer TOUTES les dépendances (dev incluses, nécessaires pour le build Vite)
# --ignore-scripts évite l'exécution du postinstall electron-builder
RUN npm install --ignore-scripts
