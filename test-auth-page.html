❓ Que voyez-vous sur http://localhost:5173/ ?

🔧 Actions de dépannage

Si vous voyez la page de connexion :

C'est NORMAL ! L'application fonctionne correctement. Vous devez :

  1. Créer un compte : Cliquez sur "Créer un nouveau compte"
  2. Ou vous connecter : Si vous avez déjà un compte, sélectionnez-le et entrez votre mot de passe

Pour créer un compte rapidement :

1. Sur http://localhost:5173/, cliquez sur "Créer un nouveau compte"
2. Remplissez le formulaire d'inscription (onboarding)
3. Une fois terminé, vous accéderez à l'application

Pour contourner l'authentification (développement) :

Ouvrez la console du navigateur (F12) et exécutez :

// Créer un utilisateur de test
const testUser = {
    id: 'test-user-' + Date.now(),
    name: 'Utilisateur Test',
    email: 'test@example.com',
    password: 'test123',
    createdAt: new Date(),
    lastLogin: new Date()
};

// Sauvegarder dans IndexedDB
db.users.add(testUser).then(() => {
    db.setCurrentUser(testUser.id).then(() => {
        console.log('✅ Utilisateur de test créé et connecté');
        location.reload();
    });
});
