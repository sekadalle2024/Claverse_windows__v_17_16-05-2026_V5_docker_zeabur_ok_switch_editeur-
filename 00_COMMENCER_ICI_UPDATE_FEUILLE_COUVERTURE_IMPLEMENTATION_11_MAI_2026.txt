Le script va:

    1. ✅ Créer un backup automatique de DemarrerMenu.tsx
    
    2. ✅ Localiser la section E-revision > PLANIFICATION
    
    3. ✅ Trouver l'étape "Feuille de couverture Implementation"
    
    4. ✅ Remplacer les modes Normal et Demo avec les nouvelles variables:
    
        Mode [Normal]:
          • [Command] = Couverture
          • [Processus] = Controle des rapprochements bancaires
          • [Contrôle] = vérifier que les rapprochements bancaires...
          • [Integration] = Implementation_programme_controle
          • [Nb de lignes] = 25
    
        Mode [Demo]:
          • Toutes les variables du mode Normal
          • PLUS: [Demo] = Activate
    
    5. ✅ Afficher un résumé détaillé des modifications
