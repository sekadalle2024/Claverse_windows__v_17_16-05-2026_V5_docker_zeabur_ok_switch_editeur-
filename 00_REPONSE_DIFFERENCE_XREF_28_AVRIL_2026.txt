📅 Date: 28 avril 2026
🎯 Sujet: Différence entre "Ouvrir X-Ref" et "Afficher X-Ref"

═══════════════════════════════════════════════════════════════════

🔍 VOTRE QUESTION
═══════════════════════════════════════════════════════════════════

"Nous avons observé deux actions ajoutées dans la section papier de 
travail du menu contextuel géré par menu.js:
- Ouvrir X-ref documentaire
- Afficher X-ref documentaire

Quelle est la différence et le rendu entre ces deux actions ?"

═══════════════════════════════════════════════════════════════════

✅ RÉPONSE SIMPLE
═══════════════════════════════════════════════════════════════════

📋 AFFICHER X-REF DOCUMENTAIRE
-------------------------------
→ Affiche TOUTE la table X-Ref dans une barre latérale
→ Pas besoin de sélectionner une ligne
→ Vue d'ensemble de tous les documents
→ Ne contacte PAS Google Drive
→ Lecture seule de la table

📂 OUVRIR X-REF DOCUMENTAIRE (Ctrl+Shift+O)
--------------------------------------------
→ Ouvre UN document spécifique depuis Google Drive
→ Nécessite de sélectionner une ligne dans la table
→ Construit le nom: [Cross references]-[Document]
→ Appelle le workflow n8n
→ Recherche et affiche le document dans la barre latérale

🎯 EXEMPLE CONCRET
═══════════════════════════════════════════════════════════════════

SCÉNARIO 1: Vue d'ensemble
---------------------------
1. Clic droit sur la table X-Ref
2. "📋 Afficher X-Ref documentaire"
3. → Barre latérale s'ouvre avec:
   [I1] Lead sheet trésorerie
   [I20] Proces verbal inventaire de caisse
   [I150] Balances generale
   ...

SCÉNARIO 2: Ouvrir un document
-------------------------------
1. Clic sur la ligne "[I150] Balances generale"
2. Clic droit
3. "📂 Ouvrir X-Ref documentaire"
4. → Workflow n8n recherche "[I150]-Balances generale"
5. → Document s'affiche dans la barre latérale depuis Google Drive

═══════════════════════════════════════════════════════════════════

🔧 WORKFLOW TECHNIQUE
═══════════════════════════════════════════════════════════════════

AFFICHER X-REF:
---------------
1. Recherche la table avec colonnes "Cross references" + "Document"
2. Extrait toutes les données
3. Affiche dans barre latérale React (XRefSidebar.tsx)
4. Pas d'appel externe

OUVRIR X-REF:
-------------
1. Vérifie qu'une ligne est sélectionnée
2. Lit [Cross references] et [Document] de la ligne
3. Construit: "[Cross references]-[Document]"
4. POST https://hkj0631c.rpcl.app/webhook/cross_reference
5. Reçoit fileId, fileUrl, mimeType
6. Affiche le document dans barre latérale
