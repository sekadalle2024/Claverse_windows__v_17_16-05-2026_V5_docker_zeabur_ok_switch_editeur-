                    📊 INFORMATIONS À COLLECTER
═══════════════════════════════════════════════════════════════

Dans la console, exécuter ces commandes et copier les résultats:

1. Vérifier les tables
```javascript
const tables = document.querySelectorAll('table');
console.log('Nombre de tables:', tables.length);
```

2. Vérifier les en-têtes
```javascript
tables.forEach((table, i) => {
  const headers = table.querySelectorAll('th');
  console.log(`Table ${i+1}:`, Array.from(headers).map(h => h.textContent.trim()));
});
```

3. Vérifier la table Résultat Final
```javascript
const resultatTable = document.querySelector('[data-resultat-table]');
if (resultatTable) {
  console.log('Table Résultat trouvée:', resultatTable);
  console.log('Contenu:', resultatTable.querySelector('td')?.textContent);
} else {
  console.log('❌ Table Résultat NON trouvée');
}
```

4. Vérifier les colonnes masquées
```javascript
tables.forEach((table, i) => {
  const headers = table.querySelectorAll('th');
  headers.forEach((h, j) => {
    if (h.style.display === 'none') {
      console.log(`Table ${i+1}, Colonne ${j+1} "${h.textContent}" est MASQUÉE`);
    }
  });
});
```
