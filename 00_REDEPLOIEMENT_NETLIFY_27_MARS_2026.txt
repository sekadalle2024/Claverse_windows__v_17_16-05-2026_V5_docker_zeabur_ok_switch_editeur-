================================================================================
  DÉPANNAGE RAPIDE
================================================================================

Problème : "Node.js non installé"
Solution : Installer Node.js depuis https://nodejs.org/

Problème : "Non authentifié sur Netlify"
Solution : Exécuter : netlify login

Problème : "Build échoue"
Solution : Vérifier les erreurs avec : npm run build

Problème : "Out of memory"
Solution : Redémarrer l'ordinateur

Problème : "Déploiement échoue"
Solution : Vérifier la connexion Internet et l'authentification

Problème : "Site ne se met pas à jour"
Solution : Vider le cache du navigateur (Ctrl+Shift+Del)
