🎯 OBJECTIF
─────────────────────────────────────────────────────────────────────────────

Modifier la nomenclature des fichiers exportés pour inclure le nom de l'étape
de mission extraite des tables.

🔄 FLUX DE TRAITEMENT
─────────────────────────────────────────────────────────────────────────────

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  1. Utilisateur clique "Export template Excel"                         │
│                    │                                                    │
│                    ▼                                                    │
│  2. exportTemplate() appelée                                           │
│                    │                                                    │
│                    ▼                                                    │
│  3. Collecte de toutes les tables du message                           │
│                    │                                                    │
│                    ▼                                                    │
│  4. extractEtapeMission(tables) appelée                                │
│                    │                                                    │
│                    ├─► Pour chaque table:                              │
│                    │   • Extraire les données                           │
│                    │   • Chercher colonnes "Rubrique" et "Description" │
│                    │   • Chercher ligne "Etape de mission"             │
│                    │   • Extraire la valeur                             │
│                    │                                                    │
│                    ▼                                                    │
│  5. Étape trouvée?                                                     │
│         │                                                               │
│         ├─► OUI: Construire nom avec étape                             │
│         │         E-audit_Etape-mission-{NomEtape}_{timestamp}.xlsx    │
│         │                                                               │
│         └─► NON: Utiliser fallback                                     │
│                   E-audit_Etape-mission-X_{timestamp}.xlsx             │
│                    │                                                    │
│                    ▼                                                    │
│  6. Sanitiser le nom (supprimer caractères spéciaux)                   │
│                    │                                                    │
│                    ▼                                                    │
│  7. Exporter le fichier Excel                                          │
│                    │                                                    │
│                    ▼                                                    │
│  8. Notification à l'utilisateur                                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
