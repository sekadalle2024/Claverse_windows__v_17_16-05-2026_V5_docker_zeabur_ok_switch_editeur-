📝 DÉTAILS TECHNIQUES
─────────────────────────────────────────────────────────────────────────────

Service de Notification (notificationService.ts)
────────────────────────────────────────────────
• Store de notifications en mémoire
• Système de listeners (pattern Observer)
• Auto-suppression après durée spécifiée
• 5 types de notifications:
  - addInfoNotification (bleu, 5s)
  - addSuccessNotification (vert, 5s)
  - addErrorNotification (rouge, 8s)
  - addWarningNotification (orange, 6s)
  - addCompletionNotification (vert, 10s)

Composant NotificationContainer (NotificationContainer.tsx)
────────────────────────────────────────────────────────────
• Hook useEffect pour s'abonner aux notifications
• Affichage en position fixed (top-right)
• Animation slide-in depuis la droite
• Bouton de fermeture manuel
• Support de plusieurs notifications simultanées

Styles (NotificationContainer.css)
───────────────────────────────────
• Position: fixed, top: 20px, right: 20px
• Z-index: 9999 (au-dessus de tout)
• Border-left coloré selon le type
• Animation @keyframes slideIn
• Responsive (mobile: pleine largeur)
• Dark mode support

Intégration dans App.tsx
─────────────────────────
• Import: import { NotificationContainer } from './components/NotificationContainer'
• Render: <NotificationContainer /> (toujours visible)
• Placé au début du JSX pour être au-dessus de tout

🎯 UTILISATION DANS LE CODE
─────────────────────────────────────────────────────────────────────────────

Import :
────────
import { addInfoNotification } from './services/notificationService';

Utilisation :
─────────────
// Notification d'information (5s)
addInfoNotification('Titre', 'Message', 5000);

// Notification de succès (5s)
addSuccessNotification('Succès', 'Opération réussie');

// Notification d'erreur (8s)
addErrorNotification('Erreur', 'Une erreur est survenue');

// Notification d'avertissement (6s)
addWarningNotification('Attention', 'Vérifiez vos données');

// Notification de complétion (10s)
addCompletionNotification('Terminé', 'Traitement terminé');
