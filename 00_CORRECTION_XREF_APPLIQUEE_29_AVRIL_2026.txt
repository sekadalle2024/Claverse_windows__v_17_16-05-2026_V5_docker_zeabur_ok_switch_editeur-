COMMENT TESTER
===============================================================

MÉTHODE 1 : Test direct dans Claraverse
-----------------------------------------

1. Ouvrez Claraverse
2. Affichez votre table X-Ref documentaire
3. Cliquez sur une ligne de la table
4. Faites clic droit
5. Menu "Papier de travail" > "Ouvrir X-Ref documentaire"
6. Ça devrait fonctionner !


MÉTHODE 2 : Diagnostic avec le script
-----------------------------------------

Si ça ne fonctionne pas :

1. Ouvrez Claraverse
2. Affichez la table X-Ref
3. Appuyez sur F12 (DevTools)
4. Onglet "Console"
5. Copiez-collez le script depuis :
   SCRIPT_DEBUG_CLARAVERSE_XREF.txt
6. Copiez le résultat et envoyez-le moi
