CAUSE PROBABLE
═══════════════════════════════════════════════════════════════════════════

Le message utilisateur n'est PAS passé au LLM, ou le LLM l'ignore.

Possibilités:
1. Message utilisateur non extrait du webhook
2. Message utilisateur non inclus dans le prompt
3. Prompt système trop directif (contient exemple détaillé sur Achats/BC)

SOLUTION IMMÉDIATE
═══════════════════════════════════════════════════════════════════════════

ÉTAPE 1: DÉBUGGER LA RÉCEPTION DU MESSAGE
───────────────────────────────────────────────────────────────────────────

Dans n8n, AVANT le node LLM, ajouter un node "Code":

```javascript
const webhook = $input.item.json;

console.log("=== DEBUG ===");
console.log("question:", webhook.question);
console.log("data:", webhook.data);
console.log("body:", webhook.body);
console.log("message:", webhook.message);
console.log("Full:", JSON.stringify(webhook, null, 2));

return { json: webhook };
```

→ Exécuter le workflow et vérifier les logs

ÉTAPE 2: EXTRAIRE LE MESSAGE CORRECTEMENT
───────────────────────────────────────────────────────────────────────────

```javascript
const webhook = $input.item.json;

const userMessage = webhook.question 
                 || webhook.data 
                 || webhook.body 
                 || webhook.message
                 || JSON.stringify(webhook);

console.log("Message extrait:", userMessage);

return { json: { userMessage: userMessage } };
```

ÉTAPE 3: PROMPT ULTRA-SIMPLE
───────────────────────────────────────────────────────────────────────────

```javascript
const userMessage = $input.item.json.userMessage;

const systemPrompt = `Tu es un expert en contrôle interne SYSCOHADA.

MISSION: Analyser le message utilisateur et générer des recommandations.

RÈGLES:
1. Lis le message utilisateur
2. Identifie le CYCLE (RH, Achats, Ventes, etc.)
3. Identifie l'ANOMALIE
4. Génère du contenu SPÉCIFIQUE à ce cycle et cette anomalie

FORMAT JSON:
{
  "Etape mission - Programme": [
    {"Intitule": "..."},
    {"Observation": "..."},
    {"Constat": "..."},
    {"Risque": "..."},
    {"Recommendation": "..."}
  ]
}`;

const fullPrompt = `${systemPrompt}

MESSAGE UTILISATEUR:
${userMessage}

GÉNÈRE LA RÉPONSE JSON:`;

console.log("Prompt complet:", fullPrompt);

return { json: { prompt: fullPrompt } };
```
