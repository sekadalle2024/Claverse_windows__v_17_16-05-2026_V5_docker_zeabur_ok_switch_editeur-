================================================================================
✅ CORRECTION APPLIQUÉE - ERREUR "FAILED TO FETCH"
Date: 06 Avril 2026
================================================================================

PROBLÈME:
---------
Erreur "failed to fetch" lors du traitement de la balance après les corrections
initiales pour l'export de la liasse.

CAUSE RACINE:
-------------
Les DataFrames pandas (balance_n_df, balance_n1_df) ajoutés directement à
results_liasse ne sont PAS sérialisables en JSON.

FastAPI ne peut pas convertir automatiquement les DataFrames en JSON pour
la réponse HTTP, ce qui provoque l'erreur "failed to fetch" côté frontend.

SOLUTION APPLIQUÉE:
-------------------
Utilisation d'un CACHE GLOBAL pour stocker les balances séparément de la
réponse JSON de l'API.

================================================================================
MODIFICATIONS APPLIQUÉES
================================================================================

✅ MODIFICATION 1: Ajout du cache global
   Fichier: py_backend/etats_financiers.py
   Ligne: ~25
   
   Code ajouté:
   ```python
   # Cache global pour stocker les balances (non sérialisables en JSON)
   _balance_cache = {}
   ```

✅ MODIFICATION 2: Import de datetime
   Fichier: py_backend/etats_financiers.py
   Ligne: ~7
   
   Code ajouté:
   ```python
   from datetime import datetime
   ```

✅ MODIFICATION 3: Stockage des balances dans le cache
   Fichier: py_backend/etats_financiers.py
   Ligne: ~1553
   
   Code modifié:
   ```python
   # CORRECTION PROBLÈME 1: Stocker les balances dans un cache global
   logger.info("📊 Stockage des balances dans le cache global pour l'export...")
   global _balance_cache
   _balance_cache = {
       'balance_n_df': balance_df,
       'balance_n1_df': balance_n1_df,
       'balance_n2_df': balance_n2_df if balance_n2_df is not None else None,
       'timestamp': datetime.now()
   }
   logger.info(f"✅ Balances stockées dans le cache: N={len(balance_df)} lignes")
   ```

✅ MODIFICATION 4: Récupération des balances depuis le cache
   Fichier: py_backend/export_liasse.py
   Ligne: ~450
   
   Code modifié:
   ```python
   # Récupérer les balances depuis le cache global
   from etats_financiers import _balance_cache
   
   balance_n_df = None
   balance_n1_df = None
   
   if _balance_cache:
       logger.info("📊 Récupération des balances depuis le cache global...")
       balance_n_df = _balance_cache.get('balance_n_df')
       balance_n1_df = _balance_cache.get('balance_n1_df')
   ```
