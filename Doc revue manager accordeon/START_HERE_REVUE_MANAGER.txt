🧪 TEST RAPIDE (30 SECONDES)
────────────────────────────

Ouvrir la console (F12) et copier-coller :

// Créer des données de test
const testPoints = [{
  num: 1,
  titre: "Point de test",
  statut: "ok",
  prep: "Commentaire préparateur",
  rev: "Commentaire reviewer"
}];

// Sauvegarder
localStorage.setItem('clara_revue_manager_TEST-001', JSON.stringify(testPoints));

// Vérifier
console.log(JSON.parse(localStorage.getItem('clara_revue_manager_TEST-001')));

// Nettoyer
localStorage.removeItem('clara_revue_manager_TEST-001');
