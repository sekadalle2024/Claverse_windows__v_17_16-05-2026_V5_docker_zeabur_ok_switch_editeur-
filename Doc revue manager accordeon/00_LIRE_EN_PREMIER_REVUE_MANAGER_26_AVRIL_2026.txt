❓ QUESTIONS FRÉQUENTES
───────────────────────

Q : Où sont stockées les données ?
R : Dans localStorage avec la clé clara_revue_manager_${reference}

Q : Les données persistent après rechargement ?
R : Oui, elles sont automatiquement rechargées depuis localStorage

Q : Comment ajouter un point ?
R : Cliquer sur le bouton "Ajouter un point" en bas du composant

Q : Comment supprimer un point ?
R : Cliquer sur l'icône poubelle à droite du point (confirmation requise)

Q : Comment changer le statut ?
R : Ouvrir le point et utiliser le dropdown de statut

Q : Les modifications sont-elles automatiquement sauvegardées ?
R : Non, il faut cliquer sur "Enregistrer la revue" pour sauvegarder

Q : Comment voir si j'ai des modifications non enregistrées ?
R : Un indicateur orange "Modifications non enregistrées" s'affiche

Q : Puis-je avoir plusieurs revues en même temps ?
R : Oui, chaque référence a sa propre clé localStorage
