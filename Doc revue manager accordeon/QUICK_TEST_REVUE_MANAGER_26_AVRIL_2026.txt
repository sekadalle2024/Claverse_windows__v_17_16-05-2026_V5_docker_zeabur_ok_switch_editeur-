📊 STRUCTURE DES DONNÉES
────────────────────────

Entrée n8n (table 10) :
{
  "no": "1",
  "Superviseur": "Commentaire superviseur",
  "Preparer": "Commentaire préparateur"
}

Sortie transformée (RevuePoint) :
{
  "num": 1,
  "titre": "Point de revue",
  "statut": "ok",
  "prep": "Commentaire préparateur",
  "rev": "Commentaire superviseur"
}

Clé localStorage :
clara_revue_manager_${reference}

Exemple : clara_revue_manager_Test-Caisse-001

🔍 DÉTECTION AUTOMATIQUE DU STATUT
───────────────────────────────────

Mots-clés détectés :

✓ "ok" → val, ok, satisf
⚠ "warn" → suivre, warn, attention
✗ "ko" → ajust, ko, anomal

Par défaut :
- Si contenu présent → "ok"
- Si vide → "warn"
