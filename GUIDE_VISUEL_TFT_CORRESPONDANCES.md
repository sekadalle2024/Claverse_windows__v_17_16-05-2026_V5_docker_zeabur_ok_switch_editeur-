# Guide Visuel - Correspondances TFT SYSCOHADA Révisé

## Vue d'ensemble

Le Tableau des Flux de Trésorerie (TFT) est structuré en 5 grandes sections avec 25 rubriques au total.

---

## 📊 Structure Complète du TFT

### 🔵 A. TRÉSORERIE D'OUVERTURE

```
┌─────────────────────────────────────────────────────────────┐
│ ZA │ Trésorerie nette au 1er janvier                       │
│    │ = Trésorerie actif N-1 - Trésorerie passif N-1       │
│    │ Comptes: 50, 51, 52, 53, 54, 57, 58, 56              │
└─────────────────────────────────────────────────────────────┘
```

**Détail des comptes:**
- **50** : Titres de placement
- **51** : Valeurs à encaisser
- **52** : Banques
- **53** : Établissements financiers et assimilés
- **54** : Chèques postaux
- **57** : Caisse
- **58** : Régies d'avances et accréditifs
- **56** : Banques, crédits de trésorerie (PASSIF)

---

### 🟢 B. FLUX OPÉRATIONNELS (6 rubriques)

```
┌─────────────────────────────────────────────────────────────┐
│ FA │ Capacité d'Autofinancement Globale (CAFG)             │
│    │ = Résultat net + Dotations - Reprises                 │
│    │   + VNC cessions - Produits cessions - Subv. virées   │
│    │ Comptes: 13, 681, 691, 697, 851, 781, 791, 797,      │
│    │          861, 81, 82, 865                             │
├─────────────────────────────────────────────────────────────┤
│ FB │ - Variation d'actif circulant HAO (1)                 │
│    │ Comptes: 48                                           │
├─────────────────────────────────────────────────────────────┤
│ FC │ - Variation des stocks                                │
│    │ Comptes: 31, 32, 33, 34, 35, 36, 37, 38              │
├─────────────────────────────────────────────────────────────┤
│ FD │ - Variation des créances                              │
│    │ Comptes: 40, 41, 42, 43, 44, 45, 46, 47, 48          │
├─────────────────────────────────────────────────────────────┤
│ FE │ + Variation du passif circulant (1)                   │
│    │ Comptes: 40, 42, 43, 44, 45, 46, 47, 48              │
├─────────────────────────────────────────────────────────────┤
│ ZB │ FLUX OPÉRATIONNELS (Somme FA à FE)                    │
│    │ NOTE B                                                │
└─────────────────────────────────────────────────────────────┘
```

**Détail CAFG (FA):**
- **13** : Résultat net de l'exercice
- **681** : Dotations aux amortissements d'exploitation
- **691** : Dotations aux provisions d'exploitation
- **697** : Dotations aux provisions financières
- **851** : Dotations HAO
- **781** : Reprises d'amortissements d'exploitation
- **791** : Reprises de provisions d'exploitation
- **797** : Reprises de provisions financières
- **861** : Reprises HAO
- **81** : Valeur comptable des cessions d'immobilisations
- **82** : Produits de cessions d'immobilisations
- **865** : Subventions d'investissement virées au résultat

**Détail Stocks (FC):**
- **31** : Marchandises
- **32** : Matières premières et fournitures liées
- **33** : Autres approvisionnements
- **34** : Produits en cours
- **35** : Services en cours
- **36** : Produits finis
- **37** : Produits intermédiaires et résiduels
- **38** : Stocks en cours de route, en consignation ou en dépôt

## 🔍 Contrôles de Cohérence

### Contrôle 1: Équilibre des flux
```
ZG = ZB + ZC + ZF
```

### Contrôle 2: Cohérence trésorerie
```
ZH = ZA + ZG
```

### Contrôle 3: Trésorerie bilan
```
ZH (calculé) = Trésorerie Actif N - Trésorerie Passif N (bilan)
```

---

## 📝 Notes Importantes

### (1) Variations du BFR
- **Augmentation d'actif circulant** = Emploi (signe -)
- **Augmentation de passif circulant** = Ressource (signe +)
- Les variations sont calculées: Solde N - Solde N-1

### (2) Emprunts
Incluent uniquement les emprunts à long et moyen terme:
- Emprunts obligataires (161)
- Emprunts bancaires (162)
- Emprunts groupe et hors groupe (1661, 1662)

### (3) Autres dettes financières
Incluent:
- Autres emprunts et dettes assimilées (16)
- Dettes liées à des participations (18)
- Excluent les crédits de trésorerie (56)

## 🎯 Utilisation Pratique

### Pour calculer le TFT, vous avez besoin de:

1. **Balance N** (exercice en cours)
2. **Balance N-1** (exercice précédent)
3. **Résultat net** de l'exercice N

### Le fichier `correspondances_syscohada.json` permet:

✅ Mapping automatique des comptes vers les rubriques TFT
✅ Calcul automatique de tous les flux
✅ Génération du TFT au format HTML
✅ Contrôles de cohérence automatiques
✅ Export dans la liasse fiscale
