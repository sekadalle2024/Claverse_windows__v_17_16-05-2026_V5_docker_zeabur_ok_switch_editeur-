🚨 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problème 1: Timeout après 60 secondes
   Solutions:
   • Vérifier les logs de build
   • Vérifier le Dockerfile dans Github
   • Redéployer manuellement via dashboard

Problème 2: Instance démarre mais ne répond pas
   Solutions:
   • Vérifier que l'app écoute sur 0.0.0.0:5000
   • Vérifier les variables d'environnement
   • Vérifier les logs d'application

Problème 3: Instance se rendort trop vite
   Explication:
   • Normal avec instance Free
   • Sleep après 65 minutes d'inactivité
   
   Solutions:
   • Accepter le sleep mode (gratuit)
   • Upgrade vers instance payante (~$5-10/mois)
