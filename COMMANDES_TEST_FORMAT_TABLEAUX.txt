📊 VÉRIFIER LA STRUCTURE RETOURNÉE
─────────────────────────────────────────────────────────────────────────────

PowerShell (avec curl):
  curl -X POST https://t22wtwxl.rpcld.app/webhook/programme_controle_comptes `
    -H "Content-Type: application/json" `
    -d '{"message":"Programme_controle_comptes"}' | ConvertFrom-Json | ConvertTo-Json -Depth 10

Résultat attendu :
  [
    {
      "data": {
        "Tableau entete": {...},
        "Tableau processus": [...]
      }
    }
  ]

Résultat incorrect (si le problème persiste) :
  [
    {
      "json": {
        "data": {
          "Tableau entete": {...},
          "Tableau processus": [...]
        }
      }
    }
  ]

🚨 DIAGNOSTIC COMPLET
─────────────────────────────────────────────────────────────────────────────

Si le problème persiste après avoir remplacé le code :

2. Vérifier la structure retournée :

   - Si "data" n'est pas présent → Problème dans le parsing du LLM

   - Si "true" → Le code n'a pas été sauvegardé
