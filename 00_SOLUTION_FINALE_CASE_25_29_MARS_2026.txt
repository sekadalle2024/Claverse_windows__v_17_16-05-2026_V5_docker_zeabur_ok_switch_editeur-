   - "Intitule", "Observation", etc. en texte normal
   - Pas de mise en gras des en-têtes

CAUSE
───────────────────────────────────────────────────────────────────────────

Le Markdown `**Intitule**` n'était PAS rendu en HTML `<strong>Intitule</strong>`.
Le parseur Markdown du frontend ne convertissait pas le gras correctement.

SOLUTION APPLIQUÉE
═══════════════════════════════════════════════════════════════════════════

FICHIER: src/services/claraApiService.ts
FONCTION: convertHeaderTableToMarkdown()
LIGNE: ~650

AVANT:
```typescript
let md = `\n**${headerText}**\n\n`;
md += `${cellValue}\n\n`;
```

APRÈS:
```typescript
let md = `\n### ${headerText}\n\n`;
md += `${cellValue}\n\n`;
```

CHANGEMENT: Remplacement du gras (`**`) par un titre Markdown niveau 3 (`###`).

RAISON: Les titres Markdown sont mieux supportés par les parseurs que le gras.
