🆘 EN CAS DE PROBLÈME
═══════════════════════════════════════════════════════════════════════════════

❌ Erreur: "Impossible d'exécuter des scripts"
───────────────────────────────────────────────────────────────────────────────
   Solution:
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   
   Puis relancer le script


❌ Erreur: "Connection reset" ou "HTTP 408"
───────────────────────────────────────────────────────────────────────────────
   Le script va automatiquement réessayer (3 tentatives)
   
   Si échec persistant:
   1. Vérifier votre connexion Internet
   2. Attendre 5 minutes
   3. Relancer le script (il reprendra où il s'est arrêté)


❌ Erreur: "Authentication failed"
───────────────────────────────────────────────────────────────────────────────
   Vous devez vous authentifier avec GitHub:
   
   1. Aller sur: https://github.com/settings/tokens
   2. Créer un Personal Access Token (classic)
   3. Cocher: repo (toutes les permissions)
   4. Copier le token
   5. Lors du push, utiliser:
      - Username: votre nom d'utilisateur GitHub
      - Password: le token (pas votre mot de passe)


❌ Le script s'arrête à une partie
───────────────────────────────────────────────────────────────────────────────
   Pas de panique! Relancez simplement le script:
   
   .\push-claraverse-v17-10-mai-2026-final.ps1
   
   Il va automatiquement sauter les parties déjà poussées
