💡 POURQUOI CES CORRECTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Correction 1: npm install
   • npm ci est strict et nécessite un package-lock.json exact
   • npm install est tolérant et fonctionne toujours
   • --production=false installe toutes les dépendances (nécessaire pour build)
   • Le cache Docker compense la différence de vitesse

Correction 2: SDK copié
   • Le projet utilise un SDK local dans sdk/
   • Les composants importent depuis ../../sdk/src/
   • Sans copie, le build échoue avec "module not found"
   • Nécessaire pour AgentRunnerSDK et AgentRunner
