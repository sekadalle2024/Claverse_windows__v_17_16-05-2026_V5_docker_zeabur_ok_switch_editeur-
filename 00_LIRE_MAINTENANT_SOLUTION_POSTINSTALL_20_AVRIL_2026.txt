💡 POURQUOI CETTE SOLUTION FONCTIONNE
═════════════════════════════════════

Le problème:
- package.json contient: "postinstall": "electron-builder install-app-deps"
- Ce script s'exécute automatiquement après npm install
- electron-builder n'est pas disponible pendant l'installation
- Résultat: erreur "not found"

La solution:
- npm install --ignore-scripts
- Ignore TOUS les scripts postinstall
- Les dépendances sont quand même installées
- Le build Vite fonctionne normalement
- electron-builder n'est nécessaire que pour desktop, pas web
