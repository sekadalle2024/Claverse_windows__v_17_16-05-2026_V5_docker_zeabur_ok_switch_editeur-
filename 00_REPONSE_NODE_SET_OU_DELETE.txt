Le node "Set" ne peut PAS extraire les variables depuis la structure complexe
User_message car :

1. La syntaxe N8N dans "Set" est limitée pour les structures imbriquées
2. User_message est un array d'objets avec des clés dynamiques
3. Il faudrait une syntaxe très complexe et fragile

Le node "Code" est MEILLEUR car :

✅ Peut parcourir l'array User_message
✅ Peut extraire dynamiquement les clés
✅ Peut gérer les variations de noms (Message, message, Cycle, cycle, etc.)
✅ Plus flexible et maintenable
✅ Inclut des logs pour debugging

🔍 COMPARAISON : NODE SET VS NODE CODE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

NODE SET (Edit Field)
─────────────────────────────────────────────────────────────────────────

❌ Syntaxe limitée pour structures complexes
❌ Difficile à maintenir
❌ Pas de logs pour debugging
❌ Ne peut pas parcourir des arrays dynamiquement
❌ Syntaxe très longue et fragile :
   {{ $json.data[0].User_message.find(item => ...) }}

NODE CODE (JavaScript)
─────────────────────────────────────────────────────────────────────────

✅ Syntaxe JavaScript complète
✅ Facile à maintenir et modifier
✅ Logs intégrés pour debugging
✅ Peut parcourir des arrays avec for...of
✅ Gestion des variations de noms (case insensitive)
✅ Code clair et lisible

💡 EXEMPLE DE DIFFÉRENCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AVEC NODE SET (compliqué et fragile) :
─────────────────────────────────────────────────────────────────────────

user_message: {{ $json.data[0].User_message.find(item => Object.keys(item)[0].toLowerCase().includes('message'))?.[Object.keys($json.data[0].User_message.find(item => Object.keys(item)[0].toLowerCase().includes('message')))[0]] || "Message non fourni" }}

⚠️ Cette syntaxe est :
- Très longue
- Difficile à lire
- Fragile (peut casser facilement)
- Impossible à debugger

AVEC NODE CODE (simple et clair) :
─────────────────────────────────────────────────────────────────────────

for (const item of userMsgArray) {
    const key = Object.keys(item)[0];
    const value = item[key];
    
    if (key.toLowerCase().includes('message')) {
        result.user_message = value;
    }
}

✅ Cette syntaxe est :
- Courte et claire
- Facile à lire
- Robuste
- Facile à debugger avec console.log
