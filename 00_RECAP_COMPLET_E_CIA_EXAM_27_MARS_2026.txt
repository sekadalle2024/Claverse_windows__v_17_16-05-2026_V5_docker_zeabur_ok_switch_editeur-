DÉTECTION AUTOMATIQUE E-CIA EXAM
─────────────────────────────────────────────────────────────────────────────
Le système détecte automatiquement les étapes E-CIA Exam en vérifiant si
la commande contient "Cours CIA".

LOGIQUE DE SÉLECTION DES MODES :
  
  1. Si l'étape a des modes spécifiques (etape.modes)
     → Utiliser ces modes
  
  2. Sinon, vérifier si la commande contient "Cours CIA"
     → Si OUI : Utiliser ECIA_MODES (uniquement Cours)
     → Si NON : Utiliser MODES (tous les modes globaux)

AVANTAGES :
  ✓ Détection automatique sans modification de chaque test
  ✓ Basée sur le contenu de la commande
  ✓ Pas de duplication de code
  ✓ Facilement extensible


⚠️  POINTS D'ATTENTION
═══════════════════════════════════════════════════════════════════════════════

1. IMPACT SUR TOUS LES LOGICIELS
   ───────────────────────────────────────────────────────────────────────
   Le renommage du mode "Normal" en "Cours" affecte TOUS les logiciels
   qui utilisent les modes par défaut.
   
   Si d'autres logiciels doivent conserver le mode "Normal", il faudra
   créer des modes spécifiques pour ces logiciels.

2. DÉPENDANCE À "COURS CIA"
   ───────────────────────────────────────────────────────────────────────
   La détection E-CIA Exam est basée sur la présence de "Cours CIA" dans
   la commande. Si cette valeur change, la détection ne fonctionnera plus.

3. MODES SPÉCIFIQUES VS MODES GLOBAUX
   ───────────────────────────────────────────────────────────────────────
   E-CIA Exam utilise maintenant des modes spécifiques. Si de nouveaux
   modes globaux sont ajoutés, ils ne seront pas disponibles pour E-CIA
   (sauf si ECIA_MODES est mis à jour).
