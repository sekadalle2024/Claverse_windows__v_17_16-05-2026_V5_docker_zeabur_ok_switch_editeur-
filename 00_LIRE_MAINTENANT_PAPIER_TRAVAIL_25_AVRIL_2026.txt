✅ VÉRIFICATIONS À EFFECTUER
────────────────────────────

1. ✅ Détection automatique
   → Console : "📋 === PAPIER DE TRAVAIL DÉTECTÉ ==="

2. ✅ Schéma de calcul affiché
   → Ligne bleue avec formules (A), (B), (C) = (A) - (B)
   → Cellule orange pour le résultat (Ecart)

3. ✅ Totalisation automatique
   → Ligne verte "Total" en bas du tableau
   → Sommes calculées pour les colonnes monétaires

4. ✅ Cross-références horizontales
   → Ligne bleue italique avec [A01], [A02], [A03]

5. ✅ Cross-références documentaires
   → Tableau avec liens vers documents sources

🐛 EN CAS DE PROBLÈME
─────────────────────

1. Vérifier la console du navigateur (F12)
   → Rechercher "📋 === PAPIER DE TRAVAIL DÉTECTÉ ==="
   → Rechercher "🔍 === DEBUT ANALYSE REPONSE N8N ==="

2. Vérifier la structure de la réponse N8N
   → Onglet Network dans DevTools
   → Chercher la requête vers n8n
   → Vérifier la réponse JSON

3. Vérifier que "Nature de test" est présent
   → Dans la table 1
   → Avec une valeur non vide
