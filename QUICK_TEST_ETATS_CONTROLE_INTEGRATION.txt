PROBLÈME: Les états de contrôle ne s'affichent pas

SOLUTIONS:
1. Vérifier que le script EtatsControleRenderer.js est chargé
   • Console: Chercher "🔍 ÉTATS DE CONTRÔLE RENDERER V1.0"
   
2. Vérifier que les résultats contiennent les états de contrôle
   • Console: window.lastEtatsFinanciersResults.controles.etats_controle
   
3. Vérifier que l'événement est déclenché
   • Console: Chercher "🎯 Événement etat-fin:success reçu"
   
4. Tester manuellement
   • Console: EtatsControleRenderer.test()

PROBLÈME: Erreur "Composant React non disponible"

SOLUTIONS:
1. Vérifier que React et ReactDOM sont chargés
   • Console: typeof React, typeof ReactDOM
   
2. Vérifier que le composant est exporté
   • Console: typeof window.EtatsControleAccordionRenderer
   
3. Rebuild le projet
   • npm run build

PROBLÈME: Accordéons ne s'ouvrent pas

SOLUTIONS:
1. Vérifier que les événements sont attachés
   • Console: Chercher "✅ Événements des accordéons attachés"
   
2. Vérifier les classes CSS
   • Inspecter l'élément et vérifier .section-header-ec
   
3. Recharger la page
   • Ctrl+F5 (hard refresh)
