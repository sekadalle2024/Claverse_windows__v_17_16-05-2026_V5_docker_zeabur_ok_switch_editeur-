🔧 MÉTHODES CLÉS
═══════════════════════════════════════════════════════════════════════════════

1. findEcartColumnIndex(table)
   ────────────────────────────────────────────────────────────────────────
   Trouve l'index de la colonne "Ecart" dans une table
   
   Recherche: "ecart", "écart", "montant" (insensible à la casse)
   Retour: Index de la colonne (0-based) ou -1 si non trouvée
   
   Exemple:
   | Compte | Libellé | Solde BG | Solde PV | Ecart | Conclusion |
     0        1         2          3          4       5
                                              ↑
                                         Index = 4

2. calculateEmptyColumnsCount(tablePrincipale, modele)
   ────────────────────────────────────────────────────────────────────────
   Calcule le nombre de colonnes vides à ajouter AVANT les variables
   
   Formule: ecartIndex - nbVariables + 1
   
   Exemple (Rapprochement avec 3 variables):
   - Total colonnes table: 6
   - Index Ecart: 4
   - Nb variables: 3
   - Calcul: 4 - 3 + 1 = 2 colonnes vides avant
   - Colonnes après: 6 - 2 - 3 = 1 colonne vide après

3. createSchemaCalculTableDirect(tablePrincipale, natureDeTest)
   ────────────────────────────────────────────────────────────────────────
   Crée le schéma de calcul avec alignement
   
   1. Détermine le modèle selon la nature de test
   2. Calcule l'alignement (colonnes vides)
   3. Construit la table HTML
   4. Insère AU-DESSUS de la table principale

4. createCrossRefTableDirect(tablePrincipale, natureDeTest)
   ────────────────────────────────────────────────────────────────────────
   Crée la cross référence avec alignement
   
   1. Détermine le modèle selon la nature de test
   2. Calcule l'alignement (colonnes vides)
   3. Construit la table HTML
   4. Insère EN DESSOUS de la table principale
