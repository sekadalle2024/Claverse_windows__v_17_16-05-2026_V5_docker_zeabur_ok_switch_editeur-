🚀 WORKFLOW POUR CHAQUE NOUVELLE NOTE
═══════════════════════════════════════════════════════════════════════════

2. Identifier les comptes dans le plan comptable

4. Tester avec la balance démo
5. Générer automatiquement le HTML dans Tests/ (test_note_xx.html)
6. Vérifier le HTML dans le navigateur
7. Documenter dans Documentation/
8. Mettre à jour les README.md
