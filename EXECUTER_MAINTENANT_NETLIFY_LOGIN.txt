========================================
  À EXÉCUTER MAINTENANT
========================================

ÉTAPE 1: Ouvrez PowerShell
===========================
Ouvrez votre terminal PowerShell dans le dossier ClaraVerse

ÉTAPE 2: Exécutez cette commande
=================================

netlify login

ÉTAPE 3: Que va-t-il se passer ?
=================================
1. Votre navigateur va s'ouvrir automatiquement
2. Vous verrez la page de connexion Netlify
3. Connectez-vous avec votre compte (ohada.finance@gmail.com)
4. Cliquez sur "Authorize" pour autoriser l'accès
5. Le terminal affichera "You are now logged in"

ÉTAPE 4: Vérifiez la connexion
===============================

netlify status

Vous devriez voir :
- Name: [Votre nom]
- Email: ohada.finance@gmail.com
- Team: SEKA

ÉTAPE 5: Relancez le déploiement
=================================

cd deploiement-netlify
.\deploy.ps1 -Message "Deploiement v14 - 22 avril 2026"

========================================
IMPORTANT
========================================
Cette connexion n'est nécessaire qu'UNE SEULE FOIS.
Après cela, tous les futurs déploiements fonctionneront automatiquement.
