❌ PROBLÈMES POSSIBLES
─────────────────────────────────────────────────────────────────────────────

1. Fichier toujours nommé "claraverse_template_..."
   → Vérifier que le fichier menu.js a été rechargé (Ctrl+F5)
   → Vérifier les logs console pour voir si extractEtapeMission() est appelée

2. Erreur JavaScript dans la console
   → Vérifier la syntaxe de la fonction extractEtapeMission()
   → Vérifier que la fonction est bien placée dans la classe

3. Étape de mission non détectée alors qu'elle existe
   → Vérifier les logs: colonnes Rubrique/Description trouvées?
   → Vérifier l'orthographe: "Etape de mission" (avec ou sans accent)
   → Vérifier que la valeur dans Description n'est pas vide

🔧 DÉPANNAGE RAPIDE
─────────────────────────────────────────────────────────────────────────────

1. Ouvrir la console (F12)
2. Chercher les logs "[Extract Etape Mission]"
3. Vérifier les étapes:
   - Nombre de tables analysées
   - Colonnes trouvées ou non
   - Valeur extraite

4. Si aucun log "[Extract Etape Mission]":
   - La fonction n'est pas appelée
   - Vérifier que le code a été modifié correctement
   - Recharger la page (Ctrl+F5)
