ÉTAPE 2 : PRÉPARER
──────────────────

1. Créer un compte Hugging Face (gratuit)
   → https://huggingface.co/join

2. Obtenir un token (Write access)
   → https://huggingface.co/settings/tokens
   → Cliquer "New token"
   → Type : Write
   → Copier le token

3. Vérifier que Git est installé
   → Ouvrir PowerShell
   → Taper : git --version
   → Si erreur, installer Git : https://git-scm.com/downloads

ÉTAPE 3 : DÉPLOYER
──────────────────

MÉTHODE AUTOMATIQUE (RECOMMANDÉE)
──────────────────────────────────
1. Ouvrir PowerShell
2. Aller dans le dossier py_backend :
   cd py_backend

3. Exécuter le script :
   ..\Doc Hugging Face deploy\deploy-huggingface.ps1

4. Suivre les instructions à l'écran
5. Entrer votre username Hugging Face
6. Entrer votre token
7. Confirmer le déploiement

MÉTHODE MANUELLE (ALTERNATIVE)
───────────────────────────────
Voir : Doc Hugging Face deploy/COMMANDES_RAPIDES.txt

ÉTAPE 4 : VÉRIFIER
──────────────────

1. Aller sur votre Space Hugging Face
2. Attendre que le build soit terminé (5-10 minutes)
3. Vérifier que le statut est "Running"
4. Tester l'API :
   → https://[votre-username]-claraverse-backend.hf.space/health
   → https://[votre-username]-claraverse-backend.hf.space/docs
