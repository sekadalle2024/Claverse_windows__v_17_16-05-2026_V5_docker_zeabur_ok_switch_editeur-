========================================
  COMMANDES RAPIDES - RECONNEXION NETLIFY
========================================

COPIER-COLLER CES COMMANDES DANS L'ORDRE
=========================================

# 1. Se reconnecter à Netlify
netlify login

# 2. Vérifier la connexion
netlify status

# 3. Relancer le déploiement
cd deploiement-netlify
.\deploy.ps1 -Message "Mise a jour application ClaraVerse - Deploiement v14 - 22 avril 2026"

========================================

ALTERNATIVE: Déploiement direct après connexion
================================================

# Si vous êtes déjà dans le dossier deploiement-netlify
cd ..
netlify login
netlify status
cd deploiement-netlify
.\deploy.ps1 -Message "Deploiement v14 - 22 avril 2026"
