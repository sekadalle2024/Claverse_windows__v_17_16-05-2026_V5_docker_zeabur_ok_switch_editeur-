💡 HYPOTHÈSES PRINCIPALES
═══════════════════════════════════════════════════════════════════════════════

HYPOTHÈSE 1: Structure HTML différente
---------------------------------------
La table dans Claraverse pourrait avoir:
  - Pas de <thead>
  - En-têtes dans <tbody>
  - Utilisation de <td> au lieu de <th>

SOLUTION:
  Modifier le code pour chercher aussi dans tbody:
  
  const headers = Array.from(table.querySelectorAll('tr:first-child th, tr:first-child td'))


HYPOTHÈSE 2: Nom de colonne légèrement différent
-------------------------------------------------
La colonne pourrait s'appeler:
  - "Cross reference" (singulier)
  - "Xref" (sans tiret)
  - "X Ref" (avec espace)

SOLUTION:
  Ajouter plus de variantes:
  
  const hasCrossRef = headers.some(h => 
    h.includes('cross') || 
    h.includes('xref') || 
    h.includes('x-ref') || 
    h.includes('x ref')
  );


HYPOTHÈSE 3: Problème de timing
--------------------------------
Le code s'exécute avant que la table soit chargée

SOLUTION:
  Ajouter un délai ou un observer:
  
  await new Promise(resolve => setTimeout(resolve, 500));

🔧 PROPOSITION DE CORRECTION
═══════════════════════════════════════════════════════════════════════════════

VERSION AMÉLIORÉE (plus flexible):

// Chercher les en-têtes dans thead OU dans la première ligne
const headerRow = table.querySelector('thead tr') || table.querySelector('tr');
const headerCells = Array.from(headerRow.querySelectorAll('th, td'));

let crossRefIndex = -1;
let documentIndex = -1;

headerCells.forEach((cell, index) => {
  const text = cell.textContent.trim().toLowerCase();
  
  // Détection plus flexible pour "Cross references"
  if (text.includes('cross') || 
      text.includes('xref') || 
      text.includes('x-ref') || 
      text.includes('x ref') ||
      text.includes('référence')) {
    crossRefIndex = index;
  }
  
  // Détection pour "Document"
  if (text.includes('document') || 
      text.includes('doc')) {
    documentIndex = index;
  }
});
