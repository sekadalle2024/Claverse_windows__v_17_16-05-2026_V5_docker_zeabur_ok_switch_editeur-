  GUIDE RAPIDE : TEMPLATES DE TABLES (CASES 35-43)
  10 Avril 2026
═══════════════════════════════════════════════════════════════════════════════

🚀 DÉMARRAGE ULTRA-RAPIDE

1. Démarrer l'application Claraverse
2. Ouvrir le chat
3. Taper l'un des mots-clés ci-dessous
4. Le template s'affiche automatiquement

═══════════════════════════════════════════════════════════════════════════════
📋 MOTS-CLÉS DISPONIBLES
═══════════════════════════════════════════════════════════════════════════════

Template_table_unicolonne
  → Génère une table à 1 colonne vide

Template_table_simple
  → Génère une table simple avec 10 lignes et 5 colonnes (A-E)

Template_table_etape_de_mission
  → Génère une table d'en-tête + table de données

Template_table_feuille_couverture_test_audit
  → Génère une feuille de couverture complète pour tests d'audit

Template_table_frap
  → Génère un template FRAP (Feuille de Révélation et d'Analyse de Problème)

Template_table_synthèses_frap
  → Génère une synthèse des FRAP

Template_table_rapport_provisoire
  → Génère un template de rapport provisoire

Template_table_rapport_final
  → Génère un template de rapport final avec plan d'action

Template_table_suivi_recos
  → Génère un template de suivi des recommandations

═══════════════════════════════════════════════════════════════════════════════
✅ EXEMPLE D'UTILISATION
═══════════════════════════════════════════════════════════════════════════════

1. Dans le chat, taper :
   Template_table_simple

2. Le système affiche :
   | Rubrique | Description |
   |----------|-------------|
   | **Etape de mission** | audit |
   | **Reference** | table 1 |

   ### 📄 Table 1
   | No | Colonne A | Colonne B | Colonne C | Colonne D | Colonne E |
   |----|-----------|-----------|-----------|-----------|-----------|
   | 1  | -         | -         | -         | -         | -         |
   | 2  | -         | -         | -         | -         | -         |
   ...

3. Éditer les cellules directement dans le chat
