🔍 Diagnostic Page Blanche - Vérification Console

📋 Instructions pour vérifier la console

Étape 1 : Ouvrir la console

Sur http://localhost:5173/, appuyez sur F12 ou Ctrl+Shift+I

Cliquez sur l'onglet Console

Étape 2 : Chercher les erreurs

Regardez s'il y a des messages en rouge

Les erreurs courantes pour une page blanche :

❌ Uncaught SyntaxError: Unexpected token
❌ Failed to resolve module specifier
❌ Uncaught ReferenceError: React is not defined
❌ Uncaught TypeError: Cannot read properties of null
❌ Error: A component suspended while responding to synchronous input

Étape 3 : Vérifier l'onglet Network

Cliquez sur l'onglet Network

Rechargez la page (Ctrl+R)

Cherchez des fichiers en rouge (erreur 404 ou 500)

⚠️ IMPORTANT

Pendant que vous faites cela, je vais vérifier le fichier src/main.tsx qui est le point d'entrée de l'application.

Le problème est probablement là - une erreur empêche React de démarrer.

🔧 Test rapide dans la console

Dans la console, tapez ces commandes une par une :

document.getElementById('root')

→ Devrait retourner un élément div

document.getElementById('root').innerHTML

→ Devrait montrer ce qu'il y a dans le div (probablement vide)

window.React

→ Vérifie si React est chargé
