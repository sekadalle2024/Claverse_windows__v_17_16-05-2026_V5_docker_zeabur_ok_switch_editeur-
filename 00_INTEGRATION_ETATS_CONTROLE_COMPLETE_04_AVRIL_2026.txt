WORKFLOW COMPLET:

1. USER TAPE "Etat fin" DANS LE CHAT
   ↓
2. claraApiService.ts DÉTECTE LE CASE 24
   • Retourne une table avec entête "Etat_fin"
   • Affiche "📂 Sélectionnez votre fichier Balance Excel..."
   ↓
3. EtatFinAutoTrigger.js DÉTECTE LA TABLE
   • Ouvre automatiquement le dialogue de sélection de fichier
   • User sélectionne le fichier Excel (balance)
   ↓
4. ENVOI VERS LE BACKEND PYTHON
   • Endpoint: http://127.0.0.1:5000/etats-financiers/process-excel
   • Fichier encodé en base64
   ↓
5. BACKEND TRAITE LA BALANCE
   • Calcul des états financiers (Bilan, CR, TFT)
   • Calcul des 8 états de contrôle
   • Génération du HTML
   ↓
6. RÉPONSE DU BACKEND
   {
     "success": true,
     "message": "États financiers générés avec succès",
     "html": "<div class='etats-fin-container'>...</div>",
     "results": {
       "bilan_actif": {...},
       "bilan_passif": {...},
       "charges": {...},
       "produits": {...},
       "totaux": {...},
       "controles": {
         "etats_controle": {
           "etat_controle_bilan_actif": {...},
           "etat_controle_bilan_passif": {...},
           "etat_controle_compte_resultat": {...},
           "etat_controle_tft": {...},
           "etat_controle_sens_comptes": {...},
           "etat_equilibre_bilan": {...},
           "etat_controle_comptes_non_integres": {...},
           "etat_controle_sens_anormal": {...}
         }
       }
     }
   }
   ↓
7. EtatFinAutoTrigger.js AFFICHE LES RÉSULTATS
   • Remplace la table avec le HTML des états financiers
   • Stocke les résultats dans window.lastEtatsFinanciersResults
   • Déclenche l'événement 'claraverse:etat-fin:success'
   ↓
8. EtatsControleRenderer.js ÉCOUTE L'ÉVÉNEMENT
   • Détecte les états de contrôle dans window.lastEtatsFinanciersResults
   • Transforme les données au format React
   • Monte le composant EtatsControleAccordionRenderer
   ↓
9. AFFICHAGE FINAL
   • États financiers (Bilan, CR, TFT) en accordéons HTML
   • États de contrôle en accordéons React (si disponibles)
   • Interface moderne et interactive

═══════════════════════════════════════════════════════════════════════════════
  📊 LES 8 ÉTATS DE CONTRÔLE
═══════════════════════════════════════════════════════════════════════════════

1. 🏢 ÉTAT DE CONTRÔLE BILAN ACTIF
   • Total Actif (N et N-1)
   • Nombre de postes
   • Variation

2. 🏛️ ÉTAT DE CONTRÔLE BILAN PASSIF
   • Total Passif (N et N-1)
   • Nombre de postes
   • Variation

3. 💰 ÉTAT DE CONTRÔLE COMPTE DE RÉSULTAT
   • Résultat Net (N et N-1)
   • Nombre de postes
   • Variation Résultat

4. 💧 ÉTAT DE CONTRÔLE TFT
   • Trésorerie finale (N et N-1)
   • Variation trésorerie
   • Flux opérationnels

5. 🔄 ÉTAT DE CONTRÔLE SENS DES COMPTES
   • Comptes en débit/crédit
   • Totaux débit/crédit
   • Équilibre

6. ⚖️ ÉTAT D'ÉQUILIBRE BILAN
   • Total Actif vs Passif + Résultat
   • Équilibre (N et N-1)
   • Vérification de cohérence

7. ⚠️ COMPTES NON INTÉGRÉS
   • Liste des comptes non reconnus
   • Raisons et montants
   • Impact sur le total

8. 🚨 SENS ANORMAL PAR NATURE
   • Comptes avec sens anormal
   • Niveau de gravité
   • Nature comptable

  🔧 DÉTAILS TECHNIQUES
═══════════════════════════════════════════════════════════════════════════════

COMPOSANT REACT:
• Nom: EtatsControleAccordionRenderer
• Props: { etatsControle: EtatsControleData }
• State: openSections (Set<string>)
• Méthodes: toggleSection, expandAll, collapseAll, formatMontant

FORMAT DES DONNÉES:
interface EtatControlePoste {
  ref: string;
  libelle: string;
  montant_n: number;
  montant_n1: number;
}

interface EtatControle {
  titre: string;
  postes: EtatControlePoste[];
}

interface EtatsControleData {
  etat_controle_bilan_actif?: EtatControle;
  etat_controle_bilan_passif?: EtatControle;
  etat_controle_compte_resultat?: EtatControle;
  etat_controle_tft?: EtatControle;
  etat_controle_sens_comptes?: EtatControle;
  etat_equilibre_bilan?: EtatControle;
  etat_controle_comptes_non_integres?: any;
  etat_controle_sens_anormal?: any;
}

ÉVÉNEMENTS:
• 'claraverse:etat-fin:success' : Déclenché après le traitement réussi
• Contient: { filename, timestamp, results }

API GLOBALE:
• window.EtatsControleRenderer.render(parentElement, results)
• window.EtatsControleRenderer.hasEtatsControle(results)
• window.EtatsControleRenderer.test()
