Instructions pour forcer le rafraîchissement

Méthode 1: Vider le cache du navigateur

  1. Ouvrez http://localhost:5173
  2. Appuyez sur Ctrl + Shift + Delete
  3. Cochez "Images et fichiers en cache"
  4. Cliquez sur "Effacer les données"
  5. Rafraîchissez la page avec Ctrl + F5

Méthode 2: Mode navigation privée

  1. Ouvrez une fenêtre de navigation privée (Ctrl + Shift + N sur Chrome)
  2. Allez sur http://localhost:5173
  3. Testez le menu Démarrer

Méthode 3: Forcer la recompilation

Dans PowerShell, exécutez:

# Arrêter l'application
.\stop-claraverse.ps1

# Supprimer le cache de build
Remove-Item -Recurse -Force node_modules\.vite -ErrorAction SilentlyContinue
Remove-Item -Recurse -Force dist -ErrorAction SilentlyContinue

# Redémarrer
.\start-claraverse.ps1

Vérification

Une fois l'application redémarrée:

  1. Ouvrez http://localhost:5173
  2. Cliquez sur le bouton "Menu Démarrer"
  3. Naviguez vers E-revision > Phase de conclusion
  4. Vous devriez voir:
    • ✅ Recos revision des comptes
    • ✅ Recos contrôle interne comptable
    • ✅ Rapport de synthèse CAC
