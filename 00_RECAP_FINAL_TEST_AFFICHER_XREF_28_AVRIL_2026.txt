🎯 POINTS CLÉS
--------------

1. NOMENCLATURE DES FICHIERS
   Format: [Cross ref]-[Document]
   Exemple: [I30]-AFRILAND.pdf
   ⚠️  Important: Respecter exactement ce format

2. STRUCTURE GOOGLE DRIVE
   Dossier CAC/
     └── Client-Exercice/
         └── Cycle comptable/
             └── [Cross ref]-[Document].ext

3. RECHERCHE EFFICACE
   • Le workflow recherche dans TOUS les dossiers clients
   • Pas besoin de spécifier le client/exercice
   • Recherche uniquement par le nom du fichier

4. AFFICHAGE DANS CLARAVERSE
   • Barre latérale à droite
   • Support PDF, Excel, Word, Images
   • Bouton "Ouvrir dans Drive"

⚠️  POINTS D'ATTENTION
----------------------

1. DOUBLONS
   Si plusieurs clients ont un fichier avec le même nom:
   • Le workflow retourne le premier trouvé
   • Solution: Utiliser des noms uniques

2. PERMISSIONS GOOGLE DRIVE
   • Le workflow doit avoir accès en lecture
   • Les liens de partage doivent être activés
   • Vérifier les credentials dans n8n

3. NOMENCLATURE STRICTE
   • Respecter le format [Cross ref]-[Document]
   • Pas d'espaces supplémentaires
   • Pas de caractères spéciaux non supportés
