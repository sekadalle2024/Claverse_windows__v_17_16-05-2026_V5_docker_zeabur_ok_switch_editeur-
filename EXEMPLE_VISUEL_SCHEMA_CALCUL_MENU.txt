RACCOURCI CLAVIER
═══════════════════════════════════════════════════════════════════════════════

Au lieu d'utiliser le menu, vous pouvez utiliser:

1. Cliquer sur une cellule de la table principale
2. Appuyer sur: Ctrl + Shift + K
3. Le schéma est ajouté automatiquement !

ACTUALISATION DU SCHÉMA
═══════════════════════════════════════════════════════════════════════════════

Si vous modifiez la "Nature de test" dans la table 2:

AVANT:
┌────────────────────────────────────────────────────────────────┐
│ Nature de test      │ Rapprochement                           │
└────────────────────────────────────────────────────────────────┘

APRÈS MODIFICATION:
┌────────────────────────────────────────────────────────────────┐
│ Nature de test      │ Validation                              │
└────────────────────────────────────────────────────────────────┘

1. Cliquer sur la table principale
2. Menu contextuel > Papier de travail > Actualiser Schéma de calcul
3. Le schéma est régénéré avec le nouveau modèle !

ANCIEN SCHÉMA (Rapprochement):
┌────────────────────────────────────────────────────────────────┐

└────────────────────────────────────────────────────────────────┘

NOUVEAU SCHÉMA (Validation):
┌────────────────────────────────────────────────────────────────┐
│ (A) │ (B) │ (C) = (A) + (B) │ (D) │ (E) = (C) - (D) │         │
└────────────────────────────────────────────────────────────────┘


SUPPRESSION DU SCHÉMA
═══════════════════════════════════════════════════════════════════════════════

1. Cliquer sur la table principale
2. Menu contextuel > Papier de travail > Supprimer Schéma de calcul
3. Confirmer la suppression
4. Le schéma et ses données sont effacés !
