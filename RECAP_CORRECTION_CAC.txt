   ARCHITECTURE TECHNIQUE
═══════════════════════════════════════════════════════════════════════════════

BACKEND (Python):
   export_synthese_cac_final.py
   ├── Charge le template Word
   ├── Trouve les marqueurs de section
   ├── Génère les sommaires
   ├── Insère tous les contenus
   └── Retourne le fichier Word

FRONTEND (JavaScript):
   menu.js → exportSyntheseCAC()
   ├── Collecte les points d'audit
   ├── Appelle /export-synthese-cac-final
   └── Télécharge le fichier

TEMPLATE:
   Doc export rapport/template final de [Export Synthese CAC].doc
   ├── Structure de base
   ├── Marqueur: "2. OBSERVATIONS D'AUDIT"
   └── Marqueur: "3. POINTS DE CONTRÔLE INTERNE"

   - Points Recos contrôle interne comptable

   ✓ Tous les champs exportés
   ✓ Retours à la ligne corrects
