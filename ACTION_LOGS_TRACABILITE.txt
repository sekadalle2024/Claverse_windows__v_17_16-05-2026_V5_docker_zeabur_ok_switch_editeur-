ÉTAPE 1: TESTER SI LE SCRIPT SE CHARGE
───────────────────────────────────────

1. Ouvrir: TEST_FLOWISE_CHARGE.html dans votre navigateur

2. Cliquer sur les boutons de test:
   - Test 1: Script Chargé?
   - Test 2: Console Fonctionne?
   - Test 3: MutationObserver?
   - Test 4: Créer Table Test

3. Observer les résultats dans "Logs Console (Temps Réel)"

═══════════════════════════════════════════════════════════════

ÉTAPE 2: SELON LES RÉSULTATS
─────────────────────────────

CAS A: Aucun log "🚀 Initialisation"
→ Le script ne se charge PAS
→ Vérifier index.html ligne 49
→ Vérifier que Flowise.js existe à la racine

CAS B: Log "🚀 Initialisation" mais rien après
→ Le script se charge mais ne détecte pas les tables
→ Problème de MutationObserver ou de sélecteur

CAS C: Logs de détection mais pas d'appel n8n
→ Erreur dans la collecte ou l'envoi
→ Chercher les erreurs JavaScript
