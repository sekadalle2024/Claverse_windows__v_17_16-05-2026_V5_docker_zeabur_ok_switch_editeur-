  EN CAS DE PROBLÈME

❌ Erreur : "HTTP 408 Timeout"
────────────────────────────────────────────────────────────────────────────────
Solution :
1. Utiliser le script avec commits multiples
2. Ou attendre quelques minutes et réessayer
3. Ou utiliser GitHub Desktop


❌ Erreur : "Connection reset"
────────────────────────────────────────────────────────────────────────────────
Solution :
1. Vérifier votre connexion Internet
2. Attendre quelques minutes
3. Relancer le push


❌ Erreur : "cannot lock ref"
────────────────────────────────────────────────────────────────────────────────
Solution :
git fetch origin
git merge origin/master --allow-unrelated-histories
git push -u origin master


❌ Erreur : "rejected - non-fast-forward"
────────────────────────────────────────────────────────────────────────────────
Solution :
git pull origin master --allow-unrelated-histories
git push -u origin master

Solution :
