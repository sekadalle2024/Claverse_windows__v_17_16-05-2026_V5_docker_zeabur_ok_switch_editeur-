MÉTHODE 1 : Menu Contextuel (Le plus simple)
---------------------------------------------

1. Taper "Etat fin" dans le chat
2. Sélectionner votre fichier Balance Excel
3. Attendre la génération des états
4. Clic droit sur la table → "📥 Exporter liasse officielle"
5. Le fichier Excel se télécharge automatiquement

--------------------------

python generer_etats_liasse.py
