🎯 DÉPLOIEMENT EN 3 ÉTAPES :

ÉTAPE 1 : INSTALLER NETLIFY CLI
  
  npm install -g netlify-cli

ÉTAPE 2 : SE CONNECTER À NETLIFY

  netlify login

  → Une fenêtre de navigateur s'ouvrira pour l'authentification

ÉTAPE 3 : DÉPLOYER

  Option A - Déploiement automatique (recommandé) :
  
    netlify init
    
    Répondre aux questions :
    - Create & configure a new site
    - Choisir votre équipe
    - Nom du site : claraverse-app
    - Build command : npm run build
    - Publish directory : dist

  Option B - Déploiement manuel :
  
    npm run build
    netlify deploy --prod --dir=dist

🐛 DÉPANNAGE :

    rm -rf node_modules package-lock.json
    npm install
    npm run build

  Erreur "Command not found: netlify" :
    npm install -g netlify-cli --force

  Erreur "File size too large" :
    node check-deploy-size.js
