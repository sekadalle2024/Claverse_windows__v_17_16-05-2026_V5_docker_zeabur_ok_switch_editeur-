Date: 30 avril 2026

🎯 SOLUTION RECOMMANDÉE: Workflow HTTP Request avec API Google Drive
====================================================================

Cette solution utilise l'API REST de Google Drive directement via HTTP Request,
évitant les problèmes avec les nodes Google Drive de n8n.

📋 ÉTAPES DE MISE EN ŒUVRE
---------------------------

ÉTAPE 1: Obtenir un Access Token Google Drive
----------------------------------------------
1. Aller dans n8n > Credentials
2. Créer/Modifier "Google Drive OAuth2 API"
3. Utiliser ces paramètres:
   - Client ID: [VOTRE_GOOGLE_CLIENT_ID]
   - Client Secret: [VOTRE_GOOGLE_CLIENT_SECRET]
   - Scope: https://www.googleapis.com/auth/drive
   - Access Type: offline

ÉTAPE 2: Créer le workflow avec HTTP Request
---------------------------------------------
Voir le fichier: workflow_n8n_ouvrir_xref_HTTP_REQUEST.json

Structure du workflow:
1. Webhook OPTIONS (preflight CORS)
2. Webhook POST (reçoit la requête)
3. HTTP Request - Rechercher Dossier CAC
4. HTTP Request - Rechercher Fichier
5. Code - Formater la réponse
6. Respond to Webhook

ÉTAPE 3: Configuration des HTTP Request nodes
----------------------------------------------

Node "Rechercher Dossier CAC":
```json
{
  "method": "GET",
  "url": "https://www.googleapis.com/drive/v3/files",
  "authentication": "predefinedCredentialType",
  "nodeCredentialType": "googleDriveOAuth2Api",
  "sendQuery": true,
  "queryParameters": {
    "q": "name='Dossier CAC' and mimeType='application/vnd.google-apps.folder' and trashed=false",
    "fields": "files(id, name)"
  }
}
```

Node "Rechercher Fichier":
```json
{
  "method": "GET",
  "url": "https://www.googleapis.com/drive/v3/files",
  "authentication": "predefinedCredentialType",
  "nodeCredentialType": "googleDriveOAuth2Api",
  "sendQuery": true,
  "queryParameters": {
    "q": "name contains '{{ $('Webhook POST').item.json.body.index }}' and '{{ $json.files[0].id }}' in parents and trashed=false",
    "fields": "files(id, name, mimeType, webViewLink, webContentLink, thumbnailLink, iconLink, size, createdTime, modifiedTime)"
  }
}
```

ÉTAPE 4: Activer le partage des fichiers
-----------------------------------------
Pour que le frontend puisse afficher les documents:

1. Dans Google Drive, sélectionner le dossier "Dossier CAC"
2. Clic droit > Partager
3. Modifier les permissions:
   - "Toute personne disposant du lien" > "Lecteur"
4. Copier le lien de partage

Ou via API:
```json
{
  "method": "POST",
  "url": "https://www.googleapis.com/drive/v3/files/{{ $json.file.id }}/permissions",
  "body": {
    "role": "reader",
    "type": "anyone"
  }
}
```

🔧 ALTERNATIVE: Corriger le workflow existant
==============================================

Si vous préférez garder les nodes Google Drive:

1. Remplacer l'opération "search" par "list"
2. Ajouter les paramètres dans "options.queryString"
3. Vérifier que les credentials sont bien configurés

Voir: workflow_n8n_ouvrir_xref_CORRIGE.json
