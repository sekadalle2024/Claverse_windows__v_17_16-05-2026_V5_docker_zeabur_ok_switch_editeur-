PROBLÈME IDENTIFIÉ
───────────────────────────────────────────────────────────────────────────
Les tableaux s'affichaient en une seule ligne sans structure:

| Intitule | |──────────────| | Défaut de sécurisation...

Au lieu de:

┌──────────────────────────────────────────────────────────┐
│ Intitule (EN GRAS)                                       │
├──────────────────────────────────────────────────────────┤
│ Défaut de sécurisation physique des fonds...            │
└──────────────────────────────────────────────────────────┘

CAUSE
───────────────────────────────────────────────────────────────────────────
Le séparateur utilisait des caractères Unicode `─` (U+2500) au lieu de
tirets simples `-` (U+002D).

Les tableaux Markdown nécessitent des TIRETS SIMPLES pour le séparateur.

CODE INCORRECT
───────────────────────────────────────────────────────────────────────────
```typescript
let md = `\n| **${headerText}** |\n`;
md += `|${'─'.repeat(headerText.length + 6)}|\n`;  // ❌ Caractère Unicode
md += `| ${cellValue.replace(/\n\n/g, '<br><br>')} |\n\n`;
```

Génère:
```markdown
| **Intitule** |
|──────────────|  ← Caractères Unicode (ne fonctionne pas)
| Défaut de sécurisation... |
```

CODE CORRIGÉ
───────────────────────────────────────────────────────────────────────────
```typescript
let md = `\n| **${headerText}** |\n`;
md += `|${'-'.repeat(headerText.length + 6)}|\n`;  // ✅ Tirets simples
md += `| ${cellValue.replace(/\n\n/g, '<br><br>')} |\n\n`;
```

Génère:
```markdown
| **Intitule** |
|--------------|  ← Tirets simples (fonctionne)
| Défaut de sécurisation... |
```
