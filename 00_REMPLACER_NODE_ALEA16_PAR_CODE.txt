Le node "Alea16" (Edit Fields) est difficile à configurer manuellement
et impossible à importer via JSON.

✅ SOLUTION : REMPLACER PAR UN NODE CODE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Au lieu de configurer 8 champs manuellement, utilise un node "Code" JavaScript
qui fait exactement la même chose en 1 seule étape.

📋 ÉTAPES À SUIVRE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ÉTAPE 1 : Supprimer le node "Alea16"
─────────────────────────────────────────────────────────────────────────

1. Dans N8N, clique sur le node "Alea16"
2. Appuie sur la touche "Delete" ou clique sur l'icône poubelle
3. Confirme la suppression

ÉTAPE 2 : Ajouter un nouveau node "Code"
─────────────────────────────────────────────────────────────────────────

1. Clique sur le "+" entre "Extract Variables for RAG" et "Agent RAG"
2. Cherche "Code" dans la liste des nodes
3. Sélectionne "Code" (JavaScript)
4. Nomme-le "Alea Generator" ou "Alea16" (comme tu veux)

ÉTAPE 3 : Copier le code JavaScript
─────────────────────────────────────────────────────────────────────────

1. Ouvre le fichier : n8n_node_alea16_remplacement_code.js
2. Copie TOUT le code
3. Colle-le dans le node "Code" que tu viens de créer
4. Sauvegarde

ÉTAPE 4 : Connecter le node
─────────────────────────────────────────────────────────────────────────

1. Connecte "Extract Variables for RAG" → "Alea Generator" (nouveau node Code)
2. Connecte "Alea Generator" → "Agent RAG"

ÉTAPE 5 : Tester
─────────────────────────────────────────────────────────────────────────

1. Exécute le workflow
2. Vérifie l'output du node "Alea Generator" :
   
   {
     "alea": 2,
     "command": "Recos contrôle interne comptable",
     "cycle": "trésorerie",
     "test": "AA010",
     "reference": "test sur la validation du compte caisse",
     "nature_test": "Rapprochement",
     "assertion": "validité",
     "anomalie": "VOL de caisse..."
   }

3. Vérifie que l'agent RAG reçoit toutes les variables
