═══════════════════════════════════════════════════════════════
  📊 ANALYSE ATTENDUE
═══════════════════════════════════════════════════════════════

Scénario A: Aucune table détectée
──────────────────────────────────
Résultat: "Nombre total de tables: 0"
Cause: Tables non chargées ou dans un iframe
Action: Attendre le chargement ou chercher dans iframe

Scénario B: Tables sans classes CSS
──────────────────────────────────
Résultat: "Classes: (aucune classe)"
Cause: Tables génériques sans classes spécifiques
Action: Utiliser sélecteur basé sur le parent

Scénario C: Tables avec classes CSS
──────────────────────────────────
Résultat: "Classes: claraverse-table audit-table"
Cause: Sélecteur actuel n'utilise pas les classes
Action: Utiliser .claraverse-table ou .audit-table
