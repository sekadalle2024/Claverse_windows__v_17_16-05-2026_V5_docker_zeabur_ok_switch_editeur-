📝 NOTES IMPORTANTES
====================

1. NOMENCLATURE DES FICHIERS
   • Format strict: [Cross ref]-[Document]
   • Exemple: [I30]-AFRILAND.pdf
   • Pas d'espaces supplémentaires

2. STRUCTURE GOOGLE DRIVE
   • Dossier racine: "Dossier CAC"
   • Sous-dossiers: Client-Exercice
   • Cycles comptables: 10 dossiers

3. PERMISSIONS
   • Le workflow doit avoir accès en lecture
   • Les credentials Google Drive doivent être valides
   • Les liens de partage doivent être activés

4. RÉSULTAT ATTENDU
   • success: true
   • file.id: ID du fichier
   • file.webViewLink: Lien d'affichage
   • file.mimeType: Type de fichier
