🔴 PROBLÈME IDENTIFIÉ ET RÉSOLU

Erreur Render lors du build:
```
error: failed to create directory /usr/local/cargo/registry/cache/
Caused by: Read-only file system (os error 30)
💥 maturin failed
Caused by: Cargo metadata failed
```

Cause: pydantic-core 2.20.1 nécessite compilation Rust sur Python 3.14.3

═══════════════════════════════════════════════════════════

✅ SOLUTION APPLIQUÉE

Deux fichiers modifiés/créés:

1️⃣ py_backend/runtime.txt (NOUVEAU)
   → Force Python 3.11.9 au lieu de 3.14.3
   → Wheels pré-compilés disponibles

2️⃣ py_backend/requirements_render.txt (MODIFIÉ)
   → pydantic 2.6.4 (au lieu de 2.8.2)
   → pydantic-core 2.16.3 (au lieu de 2.20.1)
   → Wheels pré-compilés disponibles
