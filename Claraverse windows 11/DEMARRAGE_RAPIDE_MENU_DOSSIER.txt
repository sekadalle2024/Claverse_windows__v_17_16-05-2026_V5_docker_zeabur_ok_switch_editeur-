🚀 EN 3 ÉTAPES
═══════════════════════════════════════════════════════════

1️⃣  OUVRIR LE MENU
    → Cliquer sur l'icône 📁 dans la sidebar (à gauche)

2️⃣  EXPLORER LES TABLES
    → Cliquer sur une rubrique pour voir les tables
    → Chaque rubrique = 1 message LLM avec tables

3️⃣  ACCÉDER À UNE TABLE
    → Cliquer sur une miniature de table
    → La page défile automatiquement vers la table


✨ FONCTIONNALITÉS CLÉS
═══════════════════════════════════════════════════════════

✅ Menu coulissant sur la moitié droite de l'écran
✅ Organisation en accordéon par message
✅ Miniatures de tables avec dimensions
✅ Navigation rapide vers les tables
✅ Rafraîchissement automatique (2 secondes)
✅ Animations fluides et design moderne


🎯 UTILISATION
═══════════════════════════════════════════════════════════

OUVRIR LE MENU :
  • Clic sur 📁 dans la sidebar

FERMER LE MENU :
  • Clic sur ✕ (en haut à droite)
  • Touche Escape
  • Clic sur une miniature (fermeture auto)

NAVIGUER :
  • Clic sur rubrique → Ouvre/Ferme l'accordéon
  • Clic sur miniature → Va vers la table
