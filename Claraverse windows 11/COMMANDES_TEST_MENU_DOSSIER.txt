  menuWidth: "60%",  // Changer de 50% à 60%
}

// Modifier l'intervalle de rafraîchissement
config: {
  autoRefreshInterval: 5000,  // Changer de 2000ms à 5000ms
}

// Modifier la durée des animations
config: {
  animationDuration: 500,  // Changer de 300ms à 500ms
}


📝 NOTES IMPORTANTES
===========================================

1. Le menu détecte uniquement les messages du système LLM
   (pas les messages de l'utilisateur)

2. Les tables doivent avoir la classe CSS :
   "min-w-full border border-gray-200 dark:border-gray-700 rounded-lg"

3. Le titre de la rubrique est extrait de la première cellule
   de la première ligne (colonne Description)

4. Le rafraîchissement automatique se fait toutes les 2 secondes
   quand le menu est visible

5. Le menu a un z-index de 20000 pour être au-dessus de tout


🚀 COMMANDES DE DÉVELOPPEMENT
===========================================

// Recharger le script sans recharger la page
const script = document.createElement('script');
script.src = '/dossier-menu.js?' + Date.now();
document.body.appendChild(script);

// Détruire et recréer le menu
if (window.dossierMenu) {
  window.dossierMenu.destroy();
}
// Puis recharger le script

// Forcer un rafraîchissement
if (window.dossierMenu) {
  window.dossierMenu.refreshAccordion();
}

🎯 RÉSOLUTION DE PROBLÈMES
===========================================

PROBLÈME : Le menu ne s'ouvre pas
SOLUTION : 
- Vérifier que dossier-menu.js est chargé
- Vérifier window.dossierMenu dans la console
- Vérifier les erreurs console

PROBLÈME : Aucune table n'apparaît
SOLUTION :
- Vérifier les sélecteurs CSS
- Vérifier que les messages ont la classe correcte
- Vérifier que les tables ont la classe correcte

PROBLÈME : Le rafraîchissement ne fonctionne pas
SOLUTION :
- Vérifier que le menu est visible
- Vérifier l'intervalle dans la config
- Vérifier les logs console

PROBLÈME : L'animation est saccadée
SOLUTION :
- Réduire la durée de l'animation
- Vérifier les performances du navigateur
- Désactiver les extensions qui peuvent interférer
