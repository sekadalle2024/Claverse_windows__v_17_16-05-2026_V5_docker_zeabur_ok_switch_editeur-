📊 COMMANDES CONSOLE (après intégration)
═══════════════════════════════════════════════════════════════

// Voir les statistiques
await claraverseCommands.getStorageInfo()

// Nettoyer les anciennes données (> 7 jours)
await claraverseCommands.cleanupOld(7)

// Garder seulement les 5 tables les plus récentes
await claraverseCommands.keepOnlyRecent(5)

// Optimiser le stockage
await claraverseCommands.optimizeStorage()

// Migrer vers IndexedDB
await claraverseCommands.migrateToIndexedDB()

// Tout supprimer (ATTENTION!)
await claraverseCommands.clearAllStorage()


🔧 DÉPANNAGE
═══════════════════════════════════════════════════════════════

❓ IndexedDB non disponible ?
   → Le système bascule automatiquement sur localStorage

❓ Données non migrées ?
   → await claraverseCommands.migrateToIndexedDB()

❓ Quota toujours dépassé ?
   → await claraverseCommands.cleanupOld(3)
   → await claraverseCommands.keepOnlyRecent(5)
