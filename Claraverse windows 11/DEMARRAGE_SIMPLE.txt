📦 SCRIPTS CRÉÉS POUR VOUS :

  PowerShell (.ps1) :
    • start-frontend.ps1  → Démarre le frontend
    • start-backend.ps1   → Démarre le backend
    • start-all.ps1       → Démarre tout

  Batch (.bat) :
    • start-frontend.bat  → Démarre le frontend
    • start-backend.bat   → Démarre le backend
    • start-all.bat       → Démarre tout

═══════════════════════════════════════════════════════════════

🎯 MÉTHODE 1 : DOUBLE-CLIC (PLUS SIMPLE)

  1. Ouvrez l'explorateur Windows
  2. Naviguez vers H:\ClaraVerse
  3. Double-cliquez sur : start-all.bat

  ✅ C'est tout ! Les serveurs démarrent automatiquement.

═══════════════════════════════════════════════════════════════

🎯 MÉTHODE 2 : LIGNE DE COMMANDE

  PowerShell :
    .\start-all.ps1

  CMD :
    start-all.bat
