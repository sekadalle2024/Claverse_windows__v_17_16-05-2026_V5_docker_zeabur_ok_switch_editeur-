📋 DIAGNOSTIC RAPIDE
───────────────────────────────────────────────────────────────

Problème: Aucun log visible
  → Vérifier que la console est ouverte (F12)
  → Vérifier que le script Flowise.js est chargé

Problème: Réponse vide
  → Cherchez "❌ RÉPONSE VIDE DÉTECTÉE" dans les logs
  → Vérifier le workflow n8n (node "Respond to Webhook")

Problème: JSON invalide
  → Cherchez "❌ ÉCHEC" dans "PARSING JSON"
  → Vérifier le format de sortie du workflow n8n

Problème: Tables non collectées
  → Cherchez "Tables critères collectées: 0"
  → Vérifier les en-têtes des tables (Rubrique, Description)
