============================================
  TEST MODELISATION TABLE N8N
============================================

1. OUVRIR LA CONSOLE (F12)

2. VERIFIER LE CHARGEMENT DU MENU
   - Chercher dans la console:
   "✅ Menu contextuel ClaraVerse v9 - Accordéon chargé"
   "🚀 Sections: ... | Modélisation n8n"

3. TESTER SUR UNE TABLE
   a) Aller dans une conversation avec des tables
   b) Cliquer sur une cellule d'une table
   c) Utiliser Ctrl+M OU clic droit > Modélisation Pandas > Modélisation table n8n

4. OBSERVER LES LOGS
   🚀 [N8N Modelisation] Démarrage de l'envoi vers n8n...
   📊 [N8N Modelisation] X table(s) trouvée(s) dans la div
   📤 [N8N Modelisation] Envoi vers endpoint n8n...
   📥 [N8N Modelisation] Réponse HTTP status: 200
   ✅ [N8N Modelisation] Tables insérées avec succès

5. VERIFIER LE RESULTAT
   - Nouvelles tables générées après la table cible
   - Tables éditables (cliquer sur une cellule)
   - Notification "✅ Modélisation n8n terminée!"
