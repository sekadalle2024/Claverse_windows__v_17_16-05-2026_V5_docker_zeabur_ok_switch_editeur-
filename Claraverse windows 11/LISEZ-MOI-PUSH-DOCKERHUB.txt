⚠️ EN CAS DE PROBLÈME
═══════════════════════════════════════════════════════════════

Problème: "docker: command not found"
Solution: Lancez Docker Desktop

Problème: "unauthorized: authentication required"
Solution: Le token a peut-être expiré ou est incorrect

Problème: "denied: requested access to the resource is denied"
Solution: Vérifiez le nom du repository

Problème: Build échoue
Solution: Vérifiez que vous êtes dans le bon répertoire
