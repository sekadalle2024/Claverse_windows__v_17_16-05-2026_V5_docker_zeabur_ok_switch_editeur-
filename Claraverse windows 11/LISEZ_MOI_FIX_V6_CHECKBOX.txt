🔍 VÉRIFICATION RAPIDE
═══════════════════════════════════════════════════════════════

Dans la console (F12):

// Trouver une checkbox cochée
const cb = document.querySelector('input[type="checkbox"]:checked');

// Vérifier l'attribut
cb.hasAttribute('checked')
// → true ✅

cb.getAttribute('checked')
// → "checked" ✅


📝 SYNCHRONISATION COMPLÈTE
═══════════════════════════════════════════════════════════════

3 indicateurs maintenant synchronisés:

1. checkbox.checked = true          (Propriété JavaScript)
2. checked="checked"                (Attribut HTML)
3. data-checked="true"              (Data attribute)


🎯 POUR N8N
═══════════════════════════════════════════════════════════════

n8n peut maintenant extraire le HTML et identifier les réponses:

// Extraire le HTML
const html = table.outerHTML;

// Compter les checkboxes cochées
const count = (html.match(/checked="checked"/g) || []).length;

// Parser et trouver les réponses
const parser = new DOMParser();
const doc = parser.parseFromString(html, 'text/html');
const checkedBoxes = doc.querySelectorAll('input[checked]');
