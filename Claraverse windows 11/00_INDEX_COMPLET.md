## 🎯 Utilisation Recommandée

### Pour Démarrer le Projet
1. Retournez au dossier parent (`H:\ClaraVerse`)
2. Double-cliquez sur `start-all.bat`

### Pour Consulter la Documentation
1. Ouvrez ce dossier
2. Commencez par `00_LISEZ_MOI_EN_PREMIER.txt`
3. Puis consultez `DEMARRAGE_SIMPLE.txt`
4. Pour plus de détails : `README_DEMARRAGE.md`

### Pour Résoudre un Problème
1. Consultez `SOLUTION_FINALE_SIMPLE.md`
2. Vérifiez l'installation avec `check-install.ps1`
3. Lisez les guides de dépannage
