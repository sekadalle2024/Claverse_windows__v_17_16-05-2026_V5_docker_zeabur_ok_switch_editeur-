📋 ÉTAPE 8: DIAGNOSTIC SI PROBLÈME
───────────────────────────────────

Si le traitement s'arrête à une étape:

ÉTAPE 1 échoue:
→ La table n'a pas l'en-tête "Flowise" exact
→ Vérifier l'orthographe et la casse

ÉTAPE 2 échoue:
→ Impossible de marquer la table
→ Vérifier la structure DOM

ÉTAPE 3 échoue:
→ Pas de message utilisateur trouvé
→ Normal si pas de message avant la table

ÉTAPE 4 échoue:
→ Aucune table Rubrique+Description trouvée
→ Vérifier que vos tables ont ces en-têtes
→ Vérifier que le mot-clé est dans la colonne Description

ÉTAPE 5 échoue:
→ Erreur d'appel à n8n
→ Vérifier que n8n est démarré
→ Vérifier l'endpoint

ÉTAPE 6 échoue:
→ Réponse n8n vide ou invalide
→ Vérifier la configuration du workflow n8n
→ Vérifier "Respond to Webhook"

ÉTAPE 7 échoue:
→ Impossible d'extraire les tables de la réponse
→ Vérifier le format de la réponse n8n

ÉTAPE 8 échoue:
→ Conteneur cible introuvable
→ Vérifier la structure DOM

ÉTAPE 9 échoue:
→ Erreur d'intégration des tables
→ Vérifier les logs d'erreur JavaScript

ÉTAPE 10 échoue:
→ Erreur de suppression de la table
→ Vérifier les logs d'erreur JavaScript
