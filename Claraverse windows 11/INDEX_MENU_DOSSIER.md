## 📞 Support

### Ressources d'Aide

1. **Documentation** : Consultez les guides ci-dessus
2. **Tests** : Utilisez `public/test-dossier-menu.html`
3. **Console** : Vérifiez les logs (F12)
4. **Debug** : Utilisez `window.dossierMenu` dans la console

### Commandes Utiles

```javascript
// Vérifier l'initialisation
window.dossierMenu.initialized

// Ouvrir le menu
window.dossierMenu.showMenu()

// Rafraîchir
window.dossierMenu.refreshAccordion()

// Obtenir les messages
window.dossierMenu.getLLMMessagesWithTables()
```

## 📚 Glossaire

| Terme | Définition |
|-------|------------|
| **Menu Dossier** | Barre latérale coulissante avec accordéon pour les tables |
| **Accordéon** | Interface qui permet d'ouvrir/fermer des sections |
| **Miniature** | Aperçu cliquable d'une table |
| **Message LLM** | Message envoyé par le système d'IA |
| **Rafraîchissement** | Mise à jour automatique du contenu |
| **Slide** | Animation de glissement horizontal |
