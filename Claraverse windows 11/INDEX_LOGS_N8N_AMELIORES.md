## 🆘 Support

### Questions Fréquentes

**Q : Où voir les logs ?**
R : Console du navigateur (F12) → Onglet Console

**Q : Les logs ralentissent-ils l'application ?**
R : Non, impact minimal sur les performances

**Q : Peut-on désactiver les logs ?**
R : Oui, en modifiant la constante CONFIG dans Flowise.js

**Q : Les logs fonctionnent-ils sur mobile ?**
R : Oui, avec les outils de développement mobile
