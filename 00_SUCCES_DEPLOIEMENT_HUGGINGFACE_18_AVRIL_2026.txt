========================================================================
   ENDPOINTS DISPONIBLES
========================================================================

Le backend expose plusieurs API :

1. HEALTH CHECK
   GET /health
   Verification du statut du serveur

2. PANDAS API
   POST /upload-balance
   POST /analyze-balance
   Analyse de fichiers Excel (balances comptables)

3. PANDAS AGENT (LangChain + Gemini)
   POST /pandas-agent/query
   Agent IA pour analyse de donnees

4. GOOGLE DRIVE PDF PROXY
   GET /gdrive-proxy/pdf
   Proxy pour acces aux PDF Google Drive

5. WORD EXPORT
   POST /word-export/generate
   Generation de documents Word

6. LEAD BALANCE (Pandas Lead)
   POST /lead-balance/upload
   POST /lead-balance/analyze
   Gestion des balances principales

7. ETATS FINANCIERS SYSCOHADA
   POST /etats-financiers/generate
   Generation d'etats financiers SYSCOHADA

8. EXPORT LIASSE
   POST /export-liasse/generate
   Export de liasse fiscale complete

9. ECHANTILLONNAGE AUDIT
   POST /echantillonnage/calculate
   Calcul d'echantillons d'audit

10. EXPORT SYNTHESE CAC
    POST /export-synthese-cac/generate
    Export de synthese pour CAC

========================================================================
   INTEGRATION AVEC VOTRE FRONTEND
========================================================================

Dans votre application React/TypeScript, utilisez :

```typescript
const BACKEND_URL = "https://ohadada-claraverse-backend.hf.space";

// Exemple : Upload de balance
const uploadBalance = async (file: File) => {
  const formData = new FormData();
  formData.append('file', file);
  
  const response = await fetch(`${BACKEND_URL}/upload-balance`, {
    method: 'POST',
    body: formData
  });
  
  return await response.json();
};

// Exemple : Health Check
const checkHealth = async () => {
  const response = await fetch(`${BACKEND_URL}/health`);
  return await response.json();
};
```
