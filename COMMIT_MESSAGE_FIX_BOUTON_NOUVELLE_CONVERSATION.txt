Fix: Bouton "Nouvelle conversation" invisible dans la sidebar

🐛 Problème résolu:
Le bouton "Nouvelle conversation" (New Chat) dans la barre latérale gauche
n'apparaissait pas ou était masqué par les scripts de l'interface.

✅ Solution appliquée:

1. Modification de design_chat.js
   - Ajout d'une exception pour ne jamais masquer le bouton "New Chat"
   - Préservation du bouton même lors du masquage des sélecteurs LLM

2. Nouveau script fix-new-chat-button.js
   - Force l'affichage du bouton "New Chat"
   - Injecte des styles CSS pour garantir la visibilité
   - Observe le DOM pour réappliquer automatiquement le fix
   - Réapplique le fix toutes les secondes pour garantir la persistance

3. Intégration dans index.html
   - Chargement du script après design_chat.js
   - Ordre de chargement optimisé

📁 Fichiers modifiés:
- public/design_chat.js (exception ajoutée)
- index.html (script intégré)

📁 Fichiers créés:
