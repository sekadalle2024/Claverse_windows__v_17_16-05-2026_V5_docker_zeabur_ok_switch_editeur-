🎯 OBJECTIF ACCOMPLI
─────────────────────────────────────────────────────────────────────────────

Modifier la nomenclature des fichiers exportés via "Export template Excel"
pour inclure le nom de l'étape de mission extraite des tables.

📊 PROBLÈME RÉSOLU
─────────────────────────────────────────────────────────────────────────────

❌ AVANT:
   - Nomenclature générique: claraverse_template_2026-05-09T23-39-26.xlsx
   - Difficile d'identifier le contenu du fichier
   - Pas de lien avec l'étape de mission

✅ APRÈS:
   - Nomenclature spécifique: E-audit_Etape-mission-Leadsheet_2026-05-10T14-30-45.xlsx
   - Nom explicite avec l'étape de mission
   - Fallback automatique: E-audit_Etape-mission-X_2026-05-10T14-30-45.xlsx

🔧 MODIFICATIONS APPORTÉES
─────────────────────────────────────────────────────────────────────────────

1. ✅ Nouvelle fonction extractEtapeMission(tables)
   - Cherche dans toutes les tables du message
   - Détecte les colonnes "Rubrique" et "Description"
   - Extrait la valeur de "Etape de mission"
   - Insensible à la casse
   - Logs de diagnostic détaillés

2. ✅ Modification de exportTemplate()
   - Appelle extractEtapeMission() avant l'export
   - Construit le nom du fichier selon la nomenclature E-audit
   - Sanitise le nom (supprime caractères spéciaux, remplace espaces)
   - Fallback automatique si étape non trouvée

🔍 LOGS DE DIAGNOSTIC
─────────────────────────────────────────────────────────────────────────────

La fonction génère des logs détaillés pour faciliter le diagnostic:

✅ Recherche dans les tables:
   🔍 [Extract Etape Mission] Recherche de l'étape de mission dans 3 table(s)

✅ Détection des colonnes:
   ⏭️ Table 1: Pas de colonnes Rubrique/Description
   ✅ Table 2: Colonnes trouvées - Rubrique: col 0, Description: col 1

✅ Extraction de la valeur:
   🎯 [Extract Etape Mission] Trouvé: "Leadsheet" (Table 2, Ligne 2)

✅ Nom du fichier:
   ✅ [Export Template] Nom fichier avec étape: E-audit_Etape-mission-Leadsheet_...

⚠️ Fallback:
   ⚠️ [Extract Etape Mission] Étape de mission non trouvée dans aucune table
   ⚠️ [Export Template] Étape de mission non trouvée, nom fichier fallback: E-audit_Etape-mission-X_...
