🎯 DÉPLOYER EN 5 ÉTAPES (10 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  CRÉER UN COMPTE
    👉 https://render.com
    👉 Cliquez "Get Started"
    👉 Inscrivez-vous avec GitHub (recommandé)

2️⃣  CONNECTER GITHUB
    👉 Dashboard → "New +" → "Web Service"
    👉 "Connect account" → Autorisez Render
    👉 Sélectionnez : Back-end-python-V0_03_03_2026

3️⃣  CONFIGURER LE SERVICE
    
    Copiez-collez ces valeurs :
    
    Name:              clara-backend-production
    Region:            Frankfurt
    Branch:            main
    Root Directory:    py_backend
    Build Command:     pip install -r requirements.txt
    Start Command:     python main.py --host 0.0.0.0 --port $PORT

4️⃣  VARIABLES D'ENVIRONNEMENT
    
    Cliquez "Advanced" puis ajoutez :
    
    PYTHONUNBUFFERED          = 1
    PYTHONDONTWRITEBYTECODE   = 1
    HOST                      = 0.0.0.0

5️⃣  CHOISIR LE PLAN ET DÉPLOYER
    
    👉 Sélectionnez "Starter" ($7/mois - SANS cold start)
    👉 Cliquez "Create Web Service"
    👉 Attendez 3-5 minutes
    👉 Votre API est en ligne ! 🎉
