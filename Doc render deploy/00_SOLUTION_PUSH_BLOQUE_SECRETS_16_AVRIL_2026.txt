✅ SOLUTION 2: NETTOYER L'HISTORIQUE (RECOMMANDÉ - 10 MIN)
═══════════════════════════════════════════════════════════════════════════════

Supprimer les secrets de l'historique Git avec git-filter-repo:

# 1. Installer git-filter-repo
pip install git-filter-repo

# 2. Créer un fichier avec les patterns à supprimer
echo "Doc cross ref documentaire menu/00_CREDENTIALS_OAUTH_GOOGLE.txt" > files-to-remove.txt

# 3. Nettoyer l'historique
git filter-repo --invert-paths --paths-from-file files-to-remove.txt

# 4. Pousser vers GitHub
git push origin master --force
