🔧 NOUVELLE SECTION: RUNTIME.TXT

Pourquoi runtime.txt est maintenant obligatoire?

1. Python 3.14.3 est trop récent
   → Pas de wheels pré-compilés pour pydantic-core
   → Nécessite compilation Rust
   → Système de fichiers Render en lecture seule

2. Python 3.11.9 est stable
   → Wheels pré-compilés disponibles
   → Pas de compilation nécessaire
   → Build rapide (3-5 minutes)

3. 100% compatible
   → Tout votre code fonctionne
   → Mêmes fonctionnalités
   → Meilleure stabilité

🐛 NOUVELLE SECTION DÉPANNAGE

### Erreur: "maturin failed" ou "Cargo metadata failed"

**Cause:** Python 3.14.3 utilisé, pydantic-core nécessite Rust

**Solution:**
1. Vérifier que runtime.txt existe dans le repo GitHub
2. Contenu doit être: python-3.11.9
3. Redéployer le service

**Vérification:**
Dans les logs Render, vous devez voir:
```
==> Using Python version 3.11.9 (from runtime.txt)
```

Si vous voyez Python 3.14.3, runtime.txt est manquant ou mal configuré.
