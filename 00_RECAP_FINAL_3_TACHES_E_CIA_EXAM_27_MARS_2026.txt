═══════════════════════════════════════════════════════════════════════════════

DÉTECTION E-CIA EXAM (Tâche 2)
─────────────────────────────────────────────────────────────────────────────
Le système détecte automatiquement les étapes E-CIA Exam en vérifiant si
la commande contient "Cours CIA".

LOGIQUE DE SÉLECTION DES MODES :
  1. Si l'étape a des modes spécifiques (etape.modes)
     → Utiliser ces modes
  
  2. Sinon, vérifier si la commande contient "Cours CIA"
     → Si OUI : Utiliser ECIA_MODES (Cours + Question Qcm)
     → Si NON : Utiliser MODES (tous les modes globaux)

REMPLACEMENT AUTOMATIQUE QCM (Tâche 3)
─────────────────────────────────────────────────────────────────────────────
Quand l'utilisateur sélectionne le mode "Question Qcm" :
  1. Le système détecte que mode.id === 'qcm'
  2. Le système vérifie si la commande contient "Cours CIA"
  3. Si OUI → Remplace "[Command] = Cours CIA" par "[Command] = QCM CIA"
  4. Si NON → Utilise la commande originale
