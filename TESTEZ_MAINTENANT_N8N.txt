🎯 INDICATEUR CLÉ: TAILLE REÇUE
--------------------------------

✅ SI > 0 caractères:
   Données reçues ! Le problème est ailleurs.

❌ SI 0 caractères:
   Réponse vide ! Problème dans n8n.
   👉 Ajoutez un node "Respond to Webhook"


🔍 DIAGNOSTIC RAPIDE
--------------------

Regardez la ligne:
  📏 Taille reçue: XXX caractères

• XXX > 0 = ✅ n8n renvoie des données
• XXX = 0 = ❌ n8n ne renvoie rien


🆘 SI RÉPONSE VIDE (0 caractères)
---------------------------------

CAUSE:
  Votre workflow n8n ne contient pas de node
  "Respond to Webhook" ou il n'est pas configuré.

SOLUTION:
  1. Ouvrez votre workflow dans n8n
  2. Ajoutez un node "Respond to Webhook" à la fin
  3. Configurez-le:
     Response Body: { "output": "{{ $json.result }}" }
  4. Sauvegardez et activez le workflow


🆘 SI JSON INVALIDE
-------------------

CAUSE:
  n8n renvoie du texte brut, pas du JSON

SOLUTION:
  Dans le node "Respond to Webhook":
  • Response Mode: JSON
  • Body doit être un objet JSON valide
