📦 INSTALLATION (30 secondes)
────────────────────────────────────────────────────────────────
1. Ouvrir Chrome → chrome://extensions/
2. Activer "Mode développeur" (coin supérieur droit)
3. Cliquer "Charger l'extension non empaquetée"
4. Sélectionner le dossier "table-detector-ext"
5. ✅ Extension installée !

🎯 PREMIER TEST (1 minute)
────────────────────────────────────────────────────────────────
1. Ouvrir test-page.html dans Chrome
2. Cliquer sur l'icône 🔍 de l'extension
3. Cliquer "Scanner"
4. ✅ 3 tables détectées !

📥 TESTER L'EXPORT (30 secondes)
────────────────────────────────────────────────────────────────
1. Après le scan, cliquer "Export"
2. Choisir "HTML"
3. Ouvrir le fichier téléchargé
4. ✅ Rapport complet généré !

🚀 UTILISATION RÉELLE
────────────────────────────────────────────────────────────────
1. Aller sur une page avec des tables (chat Claraverse)
2. Cliquer sur l'icône de l'extension
3. Scanner la page
4. Copier les sélecteurs nécessaires (clic sur le sélecteur)
5. Exporter si besoin de documentation

📋 FORMATS D'EXPORT DISPONIBLES
────────────────────────────────────────────────────────────────
🌐 HTML  → Page web complète (visualisation)
📋 JSON  → Données structurées (traitement)
📄 PDF   → Document imprimable (via impression)
📝 DOC   → Format Word/RTF (édition)

🔧 DÉPANNAGE RAPIDE
────────────────────────────────────────────────────────────────
❌ Aucune table détectée ?
   → Vérifier que la page contient des <table>
   → Certains chats utilisent des <div> stylisés

❌ Bouton Export invisible ?
   → Scanner d'abord avec le bouton "Scanner"
   → Le bouton Export apparaît après un scan réussi

❌ Extension ne charge pas ?
   → Vérifier que manifest.json est dans le dossier
   → Activer le mode développeur
   → Recharger la page chrome://extensions/
