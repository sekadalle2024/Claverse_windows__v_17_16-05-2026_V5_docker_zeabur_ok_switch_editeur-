## 🔧 Configuration Git Optimale

Les scripts appliquent automatiquement:

```powershell
git config core.compression 0
git config http.postBuffer 2147483647
git config http.lowSpeedTime 999999
git config http.lowSpeedLimit 0
git config http.maxRequestBuffer 536870912
```

**Avantages**:
- Désactive la compression (plus rapide)
- Buffer HTTP de 2 GB
- Pas de timeout de vitesse
- Buffer de requête de 512 MB
